22. Differentiate a radio button from a checkbox?​


Answer:

There is a fundamental difference between them. In a checkbox group, a user can select more than one option. Each checkbox operates individually, so a user can toggle each response "on" and "off." Radio buttons, however, operate as a group and provide mutually exclusive selection values.

25. In creating an online survey, which widget should you use to allow the user to select his/her religion? Select the correct response: O Checkbox O Switch Toggle Button Radio Button​


Answer:

The correct widget to use to allow the user to select his/her religion in an online survey is Radio Button.

Radio Button is a widget in Android that allows the user to select one option from a group of options. In an online survey, the user needs to select only one religion from a list of options, and Radio Button is the appropriate widget to use for this purpose.

Checkbox, Switch, and Toggle Button are used to allow the user to select multiple options or to turn a setting on or off.
