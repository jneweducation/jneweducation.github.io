7. Keith bakes 12pies a day for 5days and then 11 more pies a day for the next 7days .How many pies does he bakes on total?​


To solve this problem, we can use basic arithmetic operations.

The total number of pies Keith bakes in the first 5 days is:

12 pies/day x 5 days = 60 pies

The total number of pies Keith bakes in the next 7 days is:

11 pies/day x 7 days = 77 pies

To find the total number of pies Keith bakes in both periods, we just need to add the number of pies from each period:

60 pies + 77 pies = 137 pies

Therefore, Keith bakes a total of 137 pies.
