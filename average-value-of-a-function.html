This function has all the values of the cell in the argument.

2. Average

This function determines the average of the values included in the argument. It calculates the sum of the cell and then divides that value by the number of cells in the argument.

3. Count

This function counts in the number of cells with numerical number in the argument. This function is useful for quickly counting item in cell range.

This function determines the highest cell value included the argument.

This function determines the lowest cell value included in the argument.

Answer:

2.This is the formula for calculating the average of a range of cells. In Excel, the function for this is =AVERAGE(range).

3.This function could refer to the COUNT or COUNTIF function, which counts the number of cells in a range that meet a certain criteria. For example, =COUNTIF(range, ">0") would count the number of cells in the range that are greater than zero.

4.This function could refer to the MAX function, which returns the highest value in a range of cells. In Excel, the function for this is =MAX(range).

5.This function could refer to the MIN function, which returns the lowest value in a range of cells. In Excel, the function for this is =MIN(range).
