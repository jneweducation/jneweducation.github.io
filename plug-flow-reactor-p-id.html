10. Reactant A (A -> R, CA0 = 26 mol/m3) passes in steady flow through four equal-size mixed flow reactors in series (t total = 2 min). When steady state is achieved the concentration of A is found to be 11, 5, 2, 1mol/m3in the four units. For this reaction, what must be t plug so as to reduce C, from CA0 = 26 to CAf= 1 mol/m3?


Answer:

To solve this problem, we need to use the concept of residence time distribution. The residence time distribution function for a system of N equal-size mixed flow reactors in series is given by:

E(t) = (1/t_total) * (t/t_total)^(N-1)

where E(t) is the fraction of fluid that has been in the system for a time t, and t_total is the total residence time.

The concentration C(t) of A as a function of time t can be found from the mass balance equation:

dC/dt = -k * C

where k is the rate constant for the reaction.

The solution to this differential equation is:

C(t) = C0 * exp(-k*t)

where C0 is the initial concentration of A.

The concentration of A at the output of each reactor is therefore:

C1 = C0 * exp(-k*t_plug)

C2 = C1 * exp(-k*t_plug)

C3 = C2 * exp(-k*t_plug)

C4 = C3 * exp(-k*t_plug)

We are given that C0 = 26 mol/m3, CAf = 1 mol/m3, C1 = 11 mol/m3, C2 = 5 mol/m3, C3 = 2 mol/m3, and C4 = 1 mol/m3.

We can use these values to form a system of equations:

11 = 26 * exp(-k*t_plug)

5 = 11 * exp(-k*t_plug)

2 = 5 * exp(-k*t_plug)

1 = 2 * exp(-k*t_plug)

Solving this system of equations for k and t_plug, we get:

k = 1.219 min^-1

t_plug = 0.570 min

Therefore, the plug flow reactor needs to have a residence time of 0.570 min to reduce the concentration of A from CA0 = 26 mol/m3 to CAf = 1 mol/m3.
