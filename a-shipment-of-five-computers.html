3. PAHELP PO A shipment of five computers contains that are slightly defective. If a retailer receives three of these computers at random, list the elements of the sample space S using the letters D and N for defective and non-defective computers, respectively. To each sample point, assign a value x of the random variable X representing the number of computers purchased by the retailer which are slightly defective. PAHELP NAMAN PO


Answer:

Step-by-step explanation:

There are 5 elements, 2 of which are D, 3 are N. It is important to note that the set consisting of three elements like {NDD} and {DND} are the same because it has 2 defective and 1 non-defective computers. We can find out how many 3-set elements can we have by listing all the possible scenarios. We can also divide this problem by isolating one variable and then using cases to all the other scenarios. We add all of the possible scenarios to determine the solution set.

We can choose to isolate defective units

Case 1: There are no defective units.

That means that all 3 are non-defective: {NNN}

There is only 1 scenario for the first case.

Case 2: There is 1 defective unit.

That means that 2 are non-defective: {DNN}, {NDN}, {NND}

There are 3 scenarios, It is important to know that {DNN} is the same as {NDN} because there 2 Ns and 1D.

Case 3: There are 2 defective units.

That means that only 1 is non-defective: {DDN}; {NDD}, {DND}

There are also 3 scenarios. Again, {DDN} is the same as {DND} since there are 2Ds and 1N on the set.

Case 4: There are 3 defective units.

It's impossible since there are only 2 defective units.

All in all, this is the solution set is

{NNN},  {DNN}, {NDN}, {NND},  {DDN}; {NDD}, {DND}. 7 in total.

9. In a shipment of radios, 13 were found to be defective. If 2% of the radios were faulty, how many radios were in the shipment


Answer:

We can set up an equation to solve this problem. Let X be the number of radios in the shipment. We know that 2% of the radios are faulty, or 0.02 * X = 13 defective radios. Solving for X gives us:

X = 13 / 0.02

= 650 radios

Therefore, there were 650 radios in the shipment.

Answer:

Aquaculture often requires transporting live animals from one place to another. A large shipment of the same type of fish can be delivered via a hauling tank and truck if the distance is not too great. But for long distances, or when the shipment is not large enough to justify a truck with hauling tanks, shipping fish in boxes is a feasible and more economical alternative. For international shipments, air freight in boxes is the only true option. The ornamental fish trade has a long and successful history in using air freight. Today, more and more aquaculture products such as eggs, fry and fingerlings are shipped by air freight in boxes. With modern air transportation, fish can be transported almost anywhere in the world and arrive in healthy condition.
