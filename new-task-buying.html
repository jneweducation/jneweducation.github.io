If it cost P29,078.00 to buy a new 32'' LED television, what isthe price of the old model?2. The perimeter of the rectangle is 96 when the length of a rectangle istwice the width. What are the dimensions of the rectangle?​


Answer:

1. The price of old model is 13,578

2.The Length is 64

  The Width is 32

2.The Length is 64

  The Width is 32

Step-by-step explanation:

 

Let      P= Price

      2X=Old Price

     NP=New Price

     

NP – 2X = P

29,078.00 – 2X = 15,500

              -2X=15,500 – 29,078.00

             -2X= -13,578

               -2X/-2 = -13,578/-2

                 X = 6,789

29,078.00 – 2X = 15,500

29,078.00 – 2(6,789) = 15,500

29,078.00 – 13,578 = 15,500

                  15,500 = 15,500

The price of old model is 13,578

_______________________________________________

Remember the formula for perimeter 2L+2W=P

Let X = Width

   2X = Length

2L+2W=P

2(2X) + 2(X) = 96

4X + 2X = 96

      6X = 96

     6X/6=96/6

      X = 16

2(2X) + 2(X) = 96

2(2*16) + 2(16) = 96

2(32) + 32 = 96

64 + 32 = 96

96= 96

The Length is 64

The Width is 32

2.) FORMULA OF THE PERIMETER OF RECTANGLE:

P = 2(l + w)

GIVEN:

P = 96

l = 2w

w = w

SOLUTION:

FIRST, FIND THE VALUE OF THE WIDTH:

P = 2 (l + w)

96 = 2 (2w + w)

96 = 2 (3w)

96 = 6w

96/6 = 6w/6

w = 16

the width is 16

SECOND, SUBSTITUTE THE VALUE OF WIDTH TO IDENTIFY THE VALUE OF LENGTH:

l = 2w

l = 2(16)

l = 32

the length is 32

Therefore, the dimensions of the rectangle are 16 × 32.

CHECK:

P = 2 (l + w)

96 = 2 ( 32 + 16 )

96 = 2 (48)

96 = 96
