We have to differentiate the equation only once to eliminate the arbitrary constant C.

[tex] (y-C)^2 = Cx \longrightarrow \text{Eq. 1} [/tex]

[tex] 2(y - C)y' = C [/tex]

[tex] y - C = \dfrac{C}{2y'} \longrightarrow \text{Eq. 2}[/tex]

[tex] \left(\dfrac{C}{2y'}\right)^2 = Cx [/tex]

[tex] \dfrac{C^2}{4(y')^2} = Cx [/tex]

[tex] \dfrac{C}{4(y')^2} = x [/tex]

[tex] C = 4x(y')^2 [/tex]

Now substitute to eq. 2,

[tex] y - 4x(y')^2 = \dfrac{4x(y')^2}{2y'} [/tex]

[tex] \therefore \boxed{y = 4x(y')^2 + 2xy'} [/tex]

Multiply 1 and 3 to get 3.

[tex]3Cx^{A}+\frac{\mathrm{d}}{\mathrm{d}x}(y)\times 3=6x-4 [/tex]

[tex]3Cx^{A}=6x-4-\frac{\mathrm{d}}{\mathrm{d}x}(y)\times 3 [/tex]

Multiply minus, 1 and 3 to get minus, 3.

[tex]3Cx^{A}=6x-4-3\frac{\mathrm{d}}{\mathrm{d}x}(y) [/tex]

The equation is in standard form.

[tex]3x^{A}C=6x-4 [/tex]

Divide both sides by 3, x, start superscript, A, end superscript.

[tex]\frac{3x^{A}C}{3x^{A}}=\frac{6x-4}{3x^{A}} [/tex]

Dividing by 3, x, start superscript, A, end superscript undoes the multiplication by 3, x, start superscript, A, end superscript.

[tex]C=\frac{6x-4}{3x^{A}} [/tex]

Divide 6, x, minus, 4 by 3, x, start superscript, A, end superscript.

[tex]C=\frac{2\left(3x-2\right)}{3x^{A}} [/tex]
