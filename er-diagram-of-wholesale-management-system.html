5. a. Zen Furniture received the sofa set that was bought at a wholesale price of PhP 13,000. The manager, Mr. Dela Cruz, marks up all furniture by 45%. At what price should the manager sell the sofa set? Word Problem Using GRESA Method.


Answer:

the selling price of the sofa set is PhP 18,850.

Step-by-step explanation:

To solve this problem using the GRESA method, we need to first identify the given information and the problem that needs to be solved. The given information is that the wholesale price of the sofa set is PhP 13,000 and the markup is 45%. The problem that needs to be solved is finding the price at which the sofa set should be sold.

Next, we need to rephrase the problem in our own words to better understand it. We can say that we need to find the selling price of the sofa set by applying a 45% markup to its wholesale price of PhP 13,000.

To solve the problem, we need to first calculate the markup amount by multiplying the wholesale price by the markup percentage. In this case, the markup amount is 45% of PhP 13,000, which is equal to PhP 5,850. We can find this value by performing the calculation 0.45 x 13,000 = 5,850.

Next, we need to add the markup amount to the wholesale price to find the selling price. In this case, the selling price is PhP 13,000 + PhP 5,850 = PhP 18,850.

Therefore, the selling price of the sofa set is PhP 18,850. The manager, Mr. Dela Cruz, should sell the sofa set at this price.

Profit Formula and Profit Percentage

Profit Formula:

Profit = HJ – HB

Profit percentage formula:

PU = (U x 100%) : HB

Information:

U = Profit

HJ = Selling Price

HB = Purchase Price

PU = Profit Percentage (%)

Loss Formula and Loss Percentage

Loss Formula:

Loss = HB – HJ

Loss percentage formula:

PR = (R x 100%) : HB

Information:

R = Loss

HJ = Selling Price

HB = Purchase Price

PR = Percentage of Loss (%)

100 U = 45 x 13,000

100 U = 585,000

U = 5,850

Answer:

A refrigerator works by using a coolant to remove heat from the food items inside and transfer it to the surroundings. The coolant is first compressed by the compressor, which heats it up. The hot coolant is then passed through the condenser coils at the back of the fridge, where the heat is dissipated. The coolant then passes through the expansion valve, where it expands and becomes a cold gas. This cold gas is passed through the expansion coils inside the fridge, where it absorbs heat from the food items and keeps them cold.
