Answer:

1. Given:

Force (F) = 33.0N

Distance (d) = 5m

Formula: Work (W) = F x d

Solution:

W = 33.0N x 5m

W = 165.0 Joules (J)

Therefore, the shopper does 165.0J of work in pushing her grocery cart.

2. Given:

Mass (m) = 50kg

Force exerted by Kiera (F) = 30.2N

Frictional force (f) = 17N

Distance (d) = 3m

a). To find the work done by Kiera:

Formula: Work (W) = F x d

Solution:

W = 30.2N x 3m

W = 90.6 J

Therefore, Kiera does 90.6J of work in pushing the cabinet.

b). To find the work done when frictional force is present:

Formula: Work (W) = (F - f) x d

Solution:

W = (30.2N - 17N) x 3m

W = 39.6 J

Therefore, the work done when frictional force is present is 39.6J.

19. Kiera pushes a 50kg cabinet across a level floor by a horizontal force of 30.2N against a frictional force of 17N. She was able to move the bed at a distance of 3m. how much work is done by a). Kiera GIVEN: FORMULA: SOLUTION: b). If frictional force is present. GIVEN: FORMULA: SOLUTION:


Answer:

a) Work done by Kiera:

Given:

Mass of cabinet, m = 50 kg

Applied force, F = 30.2 N

Frictional force, f = 17 N

Distance moved, d = 3 m

The net force on the cabinet is the difference between the applied force and the frictional force:

Net force = F - f

Net force = 30.2 N - 17 N

Net force = 13.2 N

The work done by Kiera is the product of the net force and the distance moved:

Work done = Net force × distance

Work done = 13.2 N × 3 m

Work done = 39.6 J

Therefore, Kiera does 39.6 Joules of work in moving the cabinet.

b) Work done with frictional force present:

The work done by the net force includes the work done against the frictional force. Therefore, the work done against friction can be found by subtracting the work done by the net force from the total work done:

Work done against friction = Total work done - Work done by net force

Total work done is the work done by Kiera, which we have already calculated as 39.6 J.

Work done by the net force can be found using the formula:

Work done by net force = Net force × distance

where distance is the distance moved by the cabinet. Therefore:

Work done by net force = Net force × distance

Work done by net force = 13.2 N × 3 m

Work done by net force = 39.6 J

Substituting the values in the equation, we get:

Work done against friction = Total work done - Work done by net force
