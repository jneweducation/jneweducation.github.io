4. Give the function of the parts of computer:1. RAM2. ExPansion Cards3. Motherboard4. CPU cooler5. CPU6. PSU7. SSD8. HDD9. Keyboard10. Monitor11. Mouse12. Computer case13. Speaker​


Answer: 1. RAM (Random Access Memory) - It is a type of computer memory that stores data temporarily and provides quick access to the CPU for faster processing.

2. Expansion Cards - These are circuit boards that are inserted into slots on the motherboard to provide additional functionality such as graphics, sound, and network connectivity.

3. Motherboard - It is the main circuit board in a computer that connects all the components and peripherals.

4. CPU cooler - It is a device that is used to dissipate heat generated by the CPU to prevent overheating and damage to the processor.

5. CPU (Central Processing Unit) - It is the "brain" of the computer that processes instructions and performs calculations.

6. PSU (Power Supply Unit) - It is a device that converts AC power from the wall outlet to DC power that can be used by the computer components.

7. SSD (Solid State Drive) - It is a storage device that uses flash memory to store data and provides faster access times compared to traditional hard drives.

8. HDD (Hard Disk Drive) - It is a traditional storage device that uses spinning disks to store data.

9. Keyboard - It is an input device that allows the user to input characters and commands into the computer.

10. Monitor - It is an output device that displays the output from the computer in the form of images and text.

11. Mouse - It is an input device that allows the user to control the movement of the cursor on the screen and select objects.

12. Computer case - It is an enclosure that houses the computer components and provides protection and organization for the parts.

13. Speaker - It is an output device that produces sound or audio from the computer.

5. what is cpu? what is The importance of cpu?​


Answer:

what is cpu?

central processing unit (CPU), principal part of any digital computer system, generally composed of the main memory, control unit, and arithmetic-logic unit. ... In modern computers, the CPU is contained on an integrated circuit chip called a microprocessor.

what is The importance of cpu?

The CPU is made up of three main components, the control unit , the immediate access store and the arithmetic and logic unit .

What is cpu? Is it important?

Any computer system's brain is the Central Processing Unit (CPU). Every computer activity eventually passes via the CPU. The microprocessor, which houses today's CPUs, is a single chip.

The original computers had transistor-filled circuit boards and were housed in massive steel cabinets. They are now tiny enough to fit in even the tiniest mobile phone or MP3 player, and they may be found in a variety of everyday household items.

CPUs are in charge of carrying out the orders given to them by the software. It is the computer core logic.

The number of "cores" or processors in a single integrated circuit can be used to classify CPUs. Single-core, dual-core, and quad-core CPUs are among them.

The speed of a CPU in a computer is measured in megahertz (MHz) or gigahertz (GHz) (GHz). The amount of clock cycles the CPU can accomplish in a second is represented by this value.

6. it is also know as a computer chassic tower system unit base unit or simply case also sometimes incorrectly referred to as the CPU or hard drive​


Answer:

A computer case, also known as a computer chassis, tower, system unit, CPU (when referring to the case as a whole rather than the processor), or cabinet, is the enclosure that contains most of the components of a personal computer (usually excluding the display, keyboard, and mouse). Cases are usually constructed from steel (often SECC—steel, electrogalvanized, cold-rolled, coil) or aluminium. Plastic is sometimes used, and other materials such as glass, wood and even Lego bricks have appeared in home-built cases.
