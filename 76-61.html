10. The final exam scores of 18 students are (in increasing order): 24, 58, 61, 67, 71, 73, 76, 79, 82, 83, 85, 87, 88, 88, 92, 93, 94, 97. find the upper quartile (q3).


Given :

The final exam scores of 18 students are 24, 58, 61, 67, 71, 73, 76, 79, 82, 83, 85, 87, 88, 88, 92, 93, 94, and 97.

Ask :

What is the upper quartile (q3) of the following numbers 24, 58, 61, 67, 71, 73, 76, 79, 82, 83, 85, 87, 88, 88, 92, 93, 94, and 97.?

Theory :

- Quartiles in data can be obtained by dividing the data sequentially into four parts that have the same value.

Quartile itself consists of three kinds, namely :

>Lower quartile (Q1)

>Middle quartile/median (Q2)

>Upper quartile ( Q3)

- Quartile for Odd data

To find the quartile for even data, we can use this formula :

Let x1, x2, x3, ..., xn is meaning the data in ascending order.

[tex]Q_1 = x_{\frac{n+1}{4} } \\Q_2 = x_{\frac{n+1}{2} } \\Q_3 = x_{\frac{3(n+1)}{4} }[/tex]

- Quartile for Even data

To find quartile for even data, we can using this formula :

x1, x2, x3, ..., xn means the data in ascending order.

[tex]Q_1 = x_{\frac{n+2}{4} } \\Q_2 = (x_{\frac{n}{2} } + x_{\frac{n+2}{2} })/2\\Q_3 = x_{\frac{(3n + 2)}{4} }[/tex]

Step to a solved problem :

24, 58, 61, 67, 71, 73, 76, 79, 82, 83, 85, 87, 88, 88, 92, 93, 94, and 97

Total data is 18, because 18 is even so me used the upper quartile for even data, and we get :

[tex]Q_3 = x_{\frac{3n+2}{4} } \\Q_3 = x_{\frac{3.18+2}{4} } \\Q_3 = x_{\frac{56}{4} } \\Q_3 = x_{14} \\Q_3 = 88[/tex]
