3. Which of the following does an operating system installed in a guest virtual machine do? A. interacts directly with host machine hardware B. controls the hypervisor C. interacts with hardware through the hypervisor D. intercepts requests for an emulated hard drive


An operating system installed in a guest virtual machine might interact with hardware through the hypervisor.

OS installed in a guest virtual machine interacts with installed hardware through a thin layer of software called the virtualization layer or hypervisor.

Definition of Virtual Machine

A virtual machine is a virtual server environment that works like a computer inside a computer.

The virtual machine runs on an isolated partition on the host computer with resources such as CPU, memory, and its own operating system. This allows users to run various applications on virtual machines and use them as they normally would on those devices.

Virtual machines are also computing resources that use software instead of a physical computer. One or more virtual machines can run on one piece of hardware.

Each virtual machine will run its own operating system and function independently of other virtual machines, even if they are all running on the same device. For example, you can use a virtual machine with the Linux operating system on your Windows computer and vice versa.

Virtual machine technology is usually used in cloud server environments to accommodate various levels of needs. For example running software that requires a different operating system than hardware, or to test new applications on a system to make it more secure.
