An integer (more commonly called an int) is a number without a decimal point.
ex. any whole number. 74 69 69 63 22 -31

A float is a floating-point number, which means it is a number that has a decimal place. Floats are used when more precision is needed.
ex. 13.1415 77.9999 69.6969
a double has 2x the precision of float. In general a double has 15 to 16 decimal digits of precision, while float only has 7.

3. Explain the ff. Data types. Int Float Bool String


Int is a data type used for values that has no fractional part. For example, -1, 3, and 67.

Float is a data type used for values that has a fractional part. For example, 4.3, 5.55 and -3.14.

Bool is a data type used to determine if a condition is either true or false. For example, 5 = 4, 1 < 2, and 55=2.

String is a data type used for values that includes a set of characters. For example, "word", "5plus" and "444".

19. non-negative integer M is called the decimal reverse of a given non-negative integer N if a string containing the decimal representation of N (without leading zeros). When reversed, becomes a decimal representation of M (possibly with leading zeros). Write a C# console program with: class Solution { public int Method(int N);}


Answer:

Here is an example C# console program that defines a class Solution with a method Method that takes a non-negative integer N and returns the decimal reverse of N.

using System;

class Solution {

public int Method(int N) {

string strN = N.ToString(); // convert N to a string

char[] charN = strN.ToCharArray(); // convert the string to a char array

Array.Reverse(charN); // reverse the order of the char array

string strM = new string(charN); // convert the reversed char array to a string

int M = int.Parse(strM); // convert the string back to an int

return M; // return the decimal reverse of N

}

}

class Program {

static void Main(string[] args) {

Solution solution = new Solution();

int N = 12345;

int M = solution.Method(N);

Console.WriteLine("The decimal reverse of " + N + " is " + M);

}

}

In this program, we first convert the input integer N to a string strN using the ToString() method. We then convert the string to a char array charN using the ToCharArray() method. We use the Array.Reverse() method to reverse the order of the characters in the array, and then convert the reversed char array back to a string strM using the new string() constructor. Finally, we convert the string strM back to an integer M using the int.Parse() method and return the decimal reverse of N.

In the Main method, we create an instance of the Solution class and call the Method method with an input value of N = 12345. We then print the result to the console using the Console.WriteLine() method. The output of the program will be: "The decimal reverse of 12345 is 54321".

27. C. Given the intercepts, find the equations of a line. 1. x-int:2; y-int: 10 2. x-int:2; y-int: -63. x-int:-4 ; y-int: 8 4. x-int:1; y-int: 75. x-int:-3; y-int:-5​


Answer:

======================================

1. The slope-intercept form of a line is y = mx+b, where m is the slope and b is the y-intercept. We can use the x-intercept and y-intercept given to find the values of m and b.

x-int: 2 means that the point (2,0) is on the line.

y-int: 10 means that the point (0,10) is on the line.

Using these two points, we can find the slope of the line:

m = (y2 - y1) / (x2 - x1)

m = (10 - 0) / (0 - 2)

m = -5

Now we can use the slope and one of the points to find the y-intercept:

y = mx + b

0 = -5(2) + b

b = 10

The equation of the line is y = -5x + 10.

==================================

2. Using the same process as above:

x-int: 2 means that the point (2,0) is on the line.

y-int: -6 means that the point (0,-6) is on the line.

m = (y2 - y1) / (x2 - x1)

m = (-6 - 0) / (0 - 2)

m = 3

y = mx + b

-6 = 3(0) + b

b = -6

The equation of the line is y = 3x - 6.

=================================

3. Using the same process as above:

x-int: -4 means that the point (-4,0) is on the line.

y-int: 8 means that the point (0,8) is on the line.

m = (y2 - y1) / (x2 - x1)

m = (8 - 0) / (0 - (-4))

m = 2

y = mx + b

8 = 2(0) + b

b = 8

The equation of the line is y = 2x + 8.

=================================

4. Using the same process as above:

x-int: 1 means that the point (1,0) is on the line.

y-int: 7 means that the point (0,7) is on the line.

m = (y2 - y1) / (x2 - x1)

m = (7 - 0) / (0 - 1)

m = -7

y = mx + b

0 = -7(1) + b

b = 7

The equation of the line is y = -7x + 7.

=================≠==================

5. Using the same process as above:

x-int: -3 means that the point (-3,0) is on the line.

y-int: -5 means that the point (0,-5) is on the line.

m = (y2 - y1) / (x2 - x1)

m = (-5 - 0) / (0 - (-3))

m = 5/3

y = mx + b

-5 = (5/3)(0) + b

b = -5

The equation of the line is y = (5/3)x - 5.
