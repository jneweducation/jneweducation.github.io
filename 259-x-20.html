12. convert 259 to binary with checking​


Answer:

(259)10 =(100000011)2.

Step by step solution

Step 1: Divide (259)10 successively by 2 until the quotient is 0:

259/2 = 129, remainder is 1

129/2 = 64, remainder is 1

64/2 = 32, remainder is 0

32/2 = 16, remainder is 0

16/2 = 8, remainder is 0

8/2 = 4, remainder is 0

4/2 = 2, remainder is 0

2/2 = 1, remainder is 0

1/2 = 0, remainder is 1

Step 2: Read from the bottom (MSB) to top (LSB) as 100000011.

So, 100000011 is the binary equivalent of decimal number 259 (Answer).

To find the difference between two numbers, subtract the number with the smallest value from the number with the largest value. The product of this sum is the difference between the two numbers.
