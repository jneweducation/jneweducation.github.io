PA BRAINLIEST!

Narito ang table na may kasama ang mga hinahanap na bilang:

| | Principal | Time | Rate | Interest |

|----------|-----------|-------|---------|----------|

| Value | 73,300 | 110.0 | 0.025% | 15,000 |

| Unknown 1 | | b | 0.5% | |

| Unknown 2 | | 3 | | |

Para mahanap ang Unknown 1 (b), kailangan nating gamitin ang formula para sa simple interest:

Interest = (Principal x Rate x Time) / 100

Substituting the values given:

15,000 = (73,300 x b x 110) / 100

Solving for b:

b = 15,000 x 100 / (73,300 x 110)

b = 0.236% (rounded to three decimal places)

Kaya naman ang sagot sa Unknown 1 (b) ay 0.236%.

Para naman sa Unknown 2, kailangan nating gamitin ang formula muli:

Interest = (Principal x Rate x Time) / 100

Substituting the values given:

15,000 = (Unknown 2 x 0.025% x 3) / 100

Solving for Unknown 2:

Unknown 2 = 15,000 x 100 / (0.025% x 3)

Unknown 2 = 20,000,000

Kaya naman ang sagot sa Unknown 2 ay 20,000,000.

30. 1. Given the data in table 2, distinguish if the given unknown sample is mixture or substance. Write your answer on the space provided.​


Answer:

Pure substances

A pure substance has a definite and constant composition — like salt or sugar. A pure substance can be either an element or a compound, but the composition of a pure substance doesn’t vary.

Elements

An element is composed of a single kind of atom. An atom is the smallest particle of an element that still has all the properties of the element.

Here’s an example: Gold is an element. If you slice and slice a chunk of gold until only one tiny particle is left that can’t be chopped any more without losing the properties that make gold gold, then you’ve got an atom.
