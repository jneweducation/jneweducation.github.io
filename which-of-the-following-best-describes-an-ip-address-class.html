1. determine which class the following IP addresses belongs to. 87.128.23.5 ________​


This IP address belongs to Class A.
Class A includes all IP addresses ranging from 0.0.0.0 to 127.255.255.255, therefore an IP address whose first octet is between 0 and 127 is an A class IP address.

Class C is a type of IP address that is used for a small network. This IP ranges between 192 to 223.


3. The IP address class which indicates that the first two octets of the IP address are the network portion.​


Answer:

As stated above, a Class A address has the first octet as the network portion and the remaining 3 octets as the host portion. Therefore, a Class A network mask is defined as 255.0. 0.0. A Class B address has the first and second octets as the network portion and the third and fourth octets as the host portion.

4. B. Exercises Exercise 1 Direction: Answer the following questions briefly and write your answer on a separate sheet of paper. 1. What is an IP address? Explain. 2. Discuss the 4 classes of IP address. 3. Why is it necessary to assign IP address? 4. Compare static IP address from a dynamic IP address patulong​


Answer:

1. An IP address, or Internet Protocol address, is a series of numbers that identifies any device on a network. Computers use IP addresses to communicate with each other both over the internet as well as on other networks.

2. In the IPv4 IP address space, there are five classes: A, B, C, D and E. Each class has a specific range of IP addresses (and ultimately dictates the number of devices you can have on your network). Primarily, class A, B, and C are used by the majority of devices on the Internet.

3. In essence, IP addresses are the identifier that allows information to be sent between devices on a network: they contain location information and make devices accessible for communication. The internet needs a way to differentiate between different computers, routers, and websites.

4. When a device is assigned a static IP address, the address does not change. Most devices use dynamic IP addresses, which are assigned by the network when they connect and change over time.
