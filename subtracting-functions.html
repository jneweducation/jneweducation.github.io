12. Explain the process of the Addition and Subtraction of Functions


Let f(x) and g(x) be two functions:

#1

Addition

Addition. We can add two functions as: (f + g)(x) = f(x) + g(x)

Example:

f(x) = 2x2 + 3x – 4 and g(x) = 2x + 3

(f + g)(x) = (2x2 + 3x – 4) + (2x + 3)

= 2x2 + 5x – 1

#2

Subtraction

We can subtract two functions as: (f – g)(x) = f(x) – g(x)

Example:

f(x) = 2x2 + 3x – 4 and g(x) = 2x + 3

(f – g)(x) = (2x2 + 3x – 4) – (2x + 3)

= 2x2 + x – 7

#3

Multiplication

(f•g)(x) = f(x)•g(x)

Example:

f(x) = 3x – 5 and g(x) = x

(f•g)(x) = (3x – 5)•(x)

= 3x2 – 5x

#4

Division

(f/g)(x) = f(x)/g(x)

Example:

f(x) = 3x2 + 4x – 3 and g(x) = x

(f/g)(x) = (3x2 + 4x – 3)/x

= 3x + 4 – 3/x
