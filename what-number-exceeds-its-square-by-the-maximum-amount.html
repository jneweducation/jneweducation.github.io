5. The sum of two numbers is 90. if 50 exceeds three seventh of the smaller number by the same amount by which the larger number exceed, Find the numbers.​


Answer:

Let's call the smaller number "x" and the larger number "y".

From the problem, we know that:

x + y = 90 (the sum of the two numbers is 90)

50 + 3/7x = y - x (50 exceeds three seventh of the smaller number by the same amount by which the larger number exceeds)

To solve for x and y, we can use substitution. Rearrange the first equation to solve for y:

y = 90 - x

Substitute this expression for y in the second equation:

50 + 3/7x = (90 - x) - x

50 + 3/7x = 90 - 2x

3/7x + 2x = 90 - 50

17/7x = 40

x = 40 * 7 / 17 = 16.47 (rounded to two decimal places)

Now that we know x, we can use the first equation to solve for y:

x + y = 90

y = 90 - x

y = 90 - 16.47 = 73.53

Therefore, the two numbers are approximately 16.47 and 73.53.

22. The square of the sum of two positive,consecutive,even numbers exceeds the sum of their squares by 336


Answer:

The two positive, consecutive, even number are 12 and 14.

Step-by-step explanation:

First step is to translate the mathematical statements given.

Since we are looking for a an even number, we can assign "n" as our first even number. Since we are looking for two even numbers that are consecutive, we can assign "n+2" as the second even number.

The square of the sum of two positive,consecutive,even numbers...

We can translate this statement as follows:

[tex](n + (n+2))^{2}[/tex] or [tex](2n+2)^{2}[/tex]

...the sum of their squares

We can translate this statement as follows:

[tex]n^{2} +(n+2)^{2}[/tex]

It was also mentioned that the square of the two numbers exceeds the sum of their squares. We can translate that as follows:

[tex](2n+2)^{2}[/tex] - 336 = [tex]n^{2} +(n+2)^{2}[/tex]

We now then solve(expand) our equation:

[tex](2n+2)^{2}[/tex] - 336 = [tex]n^{2} +(n+2)^{2}[/tex](2n+2)(2n+2) - 336 = [tex]n^{2}[/tex] + (n+2)(n+2)[tex]4n^{2} + 4n + 4 -336 = 2n^{2} + 4n +4[/tex][tex]2n^{2} + 4n -336=0[/tex]

We now use the quadratic formula to find the roots or possible values of n.

Quadratic Formula:

n = [tex]\frac{-4(plusand minus)the square root of (b^{2}-4ac )}{2a}[/tex]

where a=2; b=4;c=-336

n = [tex]\frac{-4(plus and minus)the square root of 4^{2}-4(2)(-336) }{2(2)}[/tex]

n = [tex]\frac{-4(plusand minus) the square root of 16 + 2688}{4}[/tex]

n = [tex]\frac{-4(plus and minus) the square root of 2704}{4}[/tex]

n = [tex]\frac{-4(plus and minus) 52}{4}[/tex]

Since we are looking for positive numbers, we can just get:

n = [tex]\frac{-4+52}{4}[/tex]

n = [tex]\frac{48}{4}[/tex]

n = 12 and n+2 = 12 + 2 = 14

To check:

[tex](2n+2)^{2}[/tex] = [tex]n^{2} +(n+2)^{2}[/tex]

[tex](12+14)^{2}[/tex] = [tex]12^{2} + 14^{2}[/tex]

[tex]26^{2}[/tex] = 144 + 196

676 = 340 (The square of the sum of two positive,consecutive,even numbers exceeds the sum of their squares by 336)

26. the square of a certain number exceed twice the square of another number by 1/8. the sum of their squares is 5/16 find possible pairs of numbers that statisfy this condition​


Answer:

(1/2,1/4), (-1/2,-1/4), (-1/4, 1/2), (-1/2, 1/4)

Step-by-step explanation:

Let x be the certain number and y be the another number

[tex]x^2 - 2y^2 = \frac{1}{8}[/tex]

[tex]x^2 + y^2 = \frac{5}{16}[/tex]

[tex]x^2 = \frac{5}{16} - y^2[/tex]

Subtract 1/8 to both sides in the first equation

[tex]x^2 - 2y^2 - \frac{1}{8}[/tex]            

Substitute y in for [tex]x^2[/tex]

[tex]\frac{5}{16} -y^2 - 2y^2 - \frac{1}{8}[/tex]

[tex]-3y^2 + \frac{3}{16}[/tex]

Solve this using quadratic formula, where a = -3, b = 0 and c = [tex]\frac{3}{16}[/tex]

[tex]y = \frac{0±\sqrt{0^2 -4(-3)(\frac{3}{16}) } }{2(-3)}[/tex]

[tex]y = \frac{±\sqrt{2.25} }{-6}[/tex]

[tex]y = \frac{±1.5 }{-6}[/tex]

[tex]y = \frac{1}{4} ,y = -\frac{1}{4}[/tex]

Since we already found y we can now find x

[tex]x^2 + y^2 = \frac{5}{16}[/tex]

if y = 1/4,

[tex]x^2 + (\frac{1}{4} )^2 = \frac{5}{16}[/tex]

[tex]x^2 + \frac{1}{16} = \frac{5}{16}[/tex]

[tex]x^2 = \frac{1}{4}[/tex]

Apply the square root property

[tex]x = ±\frac{1}{2}[/tex]

if y = -1/4, then the result of x is the same because -1/4^2 = 1/4^2

Now you can just change the sign of either x or y or both.
