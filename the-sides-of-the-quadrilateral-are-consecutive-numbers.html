2. The sides of a quadrilateral are consecutive numbers. If the perimeter is 160, how longs is each side?


Answer:

The sides of the quadrilateral are 38.5, 39.5, 40.5, and 41.5 long.

Step-by-Step Explanation:

Word Problem: Quadrilateral

The problem is asking about how long is each side of the quadrilateral. We know that quadrilateral is a polygon with four sides. Since the problem did not state the kind of quadrilateral involved in the problem, this means that the quadrilateral here has four different sides. To find the answer, let us analyze the given facts.

Given:

Since the sides of the quadrilateral are consecutive numbers, the lengths of the sides should be following each other continuously in the order from least to greatest. Let x be the first side.

x - first side

x + 1 - second side

x + 2 - third side

x + 3 - fourth side

160 - perimeter

Equation:

x + x + 1 + x + 2 + x + 3 = 160

Solution:

Combine similar terms.

x + x + 1 + x + 2 + x + 3 = 160

4x + 6 = 160

4x = 160 - 6

4x = 154

4x/4 = 154/4

x = 38.5

The first side is 38.5.

Plug in the value of x to find the lengths of the other sides.

Second side: x + 1 = 38.5 + 1 = 39.5

The second side is 39.5.

Third side: x + 2 = 38.5 + 2 = 40.5

The third side is 40.5.

Fourth side: x + 3 = 38.5 + 3 = 41.5

The fourth side is 41.5.

Final Answer:

38.5, 39.5, 40.5, 41.5

Checking:

Plug in the sides of the quadrilateral to see if the answers are correct.

38.5 + 39.5 + 40.5 + 41.5 = 160

160 = 160 ✔

Let the sides of the quadrilateral have lengths x, x+1, x+2, and x+3. The perimeter of the quadrilateral is then 4x + 6. Since the perimeter is 160, we have

4x + 6 = 160

Solving for x, we find that x = 36. Therefore, the lengths of the sides of the quadrilateral are 36, 37, 38, and 39.

12. 1. The sides of the quadrilateral are consecutive numbers. If the perimeter is 160,how long is each side?​


Answer:

38.5, 39.5, 40.5, 41.5

Step-by-step explanation:

x + (x+1) + (x+2) + (x+3) = 160

x + x + 1 + x + 2 + x + 3 = 160

4x + 6 = 160 {combine like terms}

4x = 160 - 6 {transpose 6}

4x/4 = 154/4 {divide both sides by 4 to isolate x}

x= 38.5 {we now have the first term as x}

x + 1= 39.5 {second term}

x + 2 = 40.5 {third term}

x + 3= 41.5 {fourth term}
