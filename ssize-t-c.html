30. Which of the following characteristics are to be considered for using T -Distribution? a. T – distribution approaches standard normal as the sample size increases b. All of the choices c. T – distribution is a family of curves based on concepts of degrees of freedom d. n < 30 #oed


Answer:

The correct options are a and c.

a. T-distribution approaches standard normal as the sample size increases. This means that as the sample size increases, the shape of the T-distribution approaches that of the standard normal distribution.

c. T-distribution is a family of curves based on concepts of degrees of freedom. The number of degrees of freedom affects the shape of the T-distribution. As the degrees of freedom increase, the T-distribution approaches the shape of the standard normal distribution.

Option b is incorrect because it includes all choices, including option d, which is not a characteristic of using T-distribution. Option d is incorrect because it suggests that T-distribution should only be used when n < 30, which is not accurate. While T-distribution is commonly used when the sample size is small (n < 30), it can also be used for larger sample sizes.
