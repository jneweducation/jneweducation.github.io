27. 116,128,105,113,116,126 order Mean Moe Median Range​


MEAN

The mean is the average of the numbers. It is easy to calculate: add up all the numbers, then divide by how many numbers there are.

Given :

116,128,105,113,116,126

Solution :

Add up all the given numbers.

116+128+105+113+116+126 = 704

Then, divide the sum equal to the number of the given in the dataset.

704 ÷ 6 = 117.33

Answer :

The Mean in the data set is 117.33

MEDIAN

The Median is the "middle" number in a list of numbers in ascending or increasing mode.

Given :

116,128,105,113,116,126

Solution :

Arrange the given numbers in ascending mode.

105,113,116,116,126,128

Then, find the Middle number in the data set.

105,113,116,116,126,128 - 116 is the middle number

Answer :

The Median in the data set is 116

MODE

The mode is the value that appears most frequently in a data set. A set of data may have one mode, more than one mode, or no mode at all.

Given :

116,128,105,113,116,126

Solution :

Find the number which is occuring the most or appears the most in the data set

105,113,116,116,126,128

The number 116 appears more frequently in the data set.

Answer :

The Mode in the data set is 116

RANGE

The range is the difference between the largest and smallest numbers.

Given :

105,113,116,116,126,128

Solution :

Find the number which is lowest and the nyumber which is highest in the dataset.

105,113,116,116,126,128

Number 105 is the lowest and 128 is the highest number.

Subtract the highest number to the lowest number.

128 - 105 = 23

Answer :

The Range of the dataset is 23
