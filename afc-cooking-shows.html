PEFI or B.P.I. calculation for broiler farm:

PEFI (Profit Efficiency Index) and BPI (Biosecurity Performance Index) are measures used to assess the efficiency and biosecurity of poultry farms, respectively. However, based on the data provided, we can only calculate the PEFI.

PEFI can be calculated using the formula:

PEFI = (Liveweight x Mortality Rate) / (FCR x Age)

Where,

Liveweight = Average liveweight of broilers

Mortality Rate = Mortality rate of broilers

FCR = Feed Conversion Ratio

Age = Age of broilers at harvest

Substituting the given values, we get:

PEFI = (1.8 x 0.05) / (1.9 x 40)

PEFI = 0.00225

Therefore, the PEFI of the broiler farm is 0.00225.

F.C.E. calculation for layer flock:

F.C.E. (Feed Conversion Efficiency) is a measure of how efficiently the birds convert their feed into eggs. It can be calculated using the formula:

F.C.E. = Feed Consumed / Egg Mass Produced

Where,

Feed Consumed = Average feed consumed per bird per day

Egg Mass Produced = Egg production per bird per day

To calculate Egg production per bird per day, we need to first convert the average egg production (AEG) to egg production per bird per day. We can do this by dividing AEG by 100 and then multiplying it by the number of days in the period (June 1 to November 30, 2021).

Number of days in the period = 183 (June 1 to November 30, 2021)

Egg production per bird per day = (AEG / 100) x (Number of days in the period / Number of birds)

Egg production per bird per day = (75 / 100) x (183 / 1)

Egg production per bird per day = 137.25

Now, we can calculate F.C.E.:

F.C.E. = AFC / Egg Mass Produced

Where,

AFC = Average feed consumed per bird per day

Egg Mass Produced = Egg production per bird per day

Substituting the given values, we get:

F.C.E. = 120 / 137.25

F.C.E. = 0.875

Therefore, the F.C.E. of the layer flock is 0.875.

Percentage Egg Production on Hen Housed Basis (HHEP) calculation:

HHEP (Hen Housed Egg Production) is a measure of the percentage of eggs produced by the birds that are housed in the flock. It can be calculated using the formula:

HHEP = (Total Egg Production / Number of Hens Housed) x 100

Where,

Total Egg Production = Total number of eggs produced in the period

Number of Hens Housed = Average number of hens housed in the period

To calculate the average number of hens housed in the period, we need to subtract the number of hens that died during the period from the average number of hens at the beginning of the period.

Average number of hens at the beginning of the period = 1000

Number of hens that died during the period = 10 + 10 + 20 = 40

Number of hens at the end of the period = 980

Average number of hens housed in the period = (Number of hens at the beginning of the period + Number of hens at the end of the period) / 2

Average number of hens housed in the period = (1000 + 980) / 2
