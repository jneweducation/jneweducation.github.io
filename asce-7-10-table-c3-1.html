To divide c^3 + 4c - 7 by c - 3, we can use long division:

```

c^2 + 3c + 10

c - 3 | c^3 + 0c^2 + 4c - 7

- c^3 + 3c^2

----------

-3c^2 + 4c

+ 3c^2 - 9c

---------

-5c + 7

```

Therefore, the quotient is c^2 + 3c + 10 and the remainder is -5c + 7. So:

c^3 + 4c - 7 = (c - 3)(c^2 + 3c + 10) - 5c + 7
