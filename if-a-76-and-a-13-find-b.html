6. Given A = 19° and a = 11, find c.

9. Given a = 7 and b = 12, find A.

165° - C = sin^-1[(37 sin 15°) / a]

C = 165° - sin^-1[(37 sin 15°) / a]

Now, using the fact that the sum of the angles in a triangle is 180°, we have:

A + B + C = 180°

15° + B + [165° - sin^-1[(37 sin 15°) / a]] = 180°

B = sin^-1[(37 sin 15°) / a] - 150°

Therefore, the value of a is needed to solve for B.
