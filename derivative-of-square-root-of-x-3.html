18. derivative of g(x)= x/1+x g’(3)=

Step-by-step explanation:

To find the derivative of the function g(x) = x/(1+x), we can use the quotient rule:

g'(x) = [(1+x) * d/dx(x) - x * d/dx(1+x)] / (1+x)^2

Simplifying this expression, we get:

g'(x) = (1+x)(1) - x(1) / (1+x)^2

g'(x) = 1 / (1+x)^2

To find g'(3), we simply plug in x=3 into the derivative formula:

g'(3) = 1 / (1+3)^2 = 1/16

Therefore, g'(3) = 1/16.
