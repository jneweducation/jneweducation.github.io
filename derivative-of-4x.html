25. Find the derivative of (3x - 2x^2)(5 + 4x)


To find the derivative of the given function, we can use the product rule of differentiation, which states that if we have two functions f(x) and g(x), then the derivative of their product f(x)g(x) is given by:

(f(x)g(x))' = f'(x)g(x) + f(x)g'(x)

Using this rule, we can find the derivative of the given function:

(3x - 2x^2)(5 + 4x) = (3x)(5) + (3x)(4x) + (-2x^2)(5) + (-2x^2)(4x)

= 15x + 12x^2 - 10x^2 - 8x^3

= -8x^3 + 2x^2 + 15x

Taking the derivative using the product rule, we get:

(-8x^3 + 2x^2 + 15x)' = (-8x^3)' + (2x^2)' + (15x)'

= -24x^2 + 4x + 15

Therefore, the derivative of the given function is -24x^2 + 4x + 15.

30. find the derivative of √5-4X


Answer:

To find the derivative of √5-4X, we need to use the power rule of differentiation and the constant multiple rule.

Recall that the power rule of differentiation states that if we have a function of the form f(x) = x^n, then its derivative is given by f'(x) = nx^(n-1).

Using the power rule, we can differentiate the term -4x as follows:

d/dx (-4x) = -4

Next, we can differentiate the term √(5) using the power rule with a fractional exponent:

d/dx (√5) = (1/2) * (d/dx (5))^(1/2) = 0

since the derivative of a constant is zero.

Therefore, the derivative of √5-4X is:

d/dx (√5-4X) = d/dx (√5) - d/dx (4X) = 0 - 4 = -4

Hence, the derivative of √5-4X is -4.
