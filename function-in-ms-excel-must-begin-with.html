The ROUND function rounds a number to a specified number of digits. For example, if cell A1 contains 23.7825, and you want to round that value to two decimal places, you can use the following formula: =ROUND(A1, 2) The result of this function is 23.78.

26. what is the parts and functions of ms excel?​


Microsoft Excel is a spreadsheet program that is widely used for organizing and analyzing data. It has many parts and functions, including:

Workbook: The main file where data is stored in Excel. A workbook can have multiple worksheets or tabs.Worksheet: A single sheet within a workbook where data is entered and analyzed.Calls: The individual rectangular boxes where data is entered in a worksheet.Rows: Horizontal lines of cells in a worksheet.Columns: Vertical lines of cells in a worksheet.Formulas: Mathematical equations used to perform calculations on data in a worksheet.Functions: Built-in formulas that perform specific calculations on data in a worksheet.Charts and Graphs: Visual representations of data that help users analyze and present data.Sorting and Filtering: Functions that allow users to organize and manipulate data based on specific criteria.Conditional Formatting: A tool that allows users to apply formatting to cells based on certain conditions, such as highlighting cells with values above a certain threshold.Pivot Tables: A tool that allows users to summarize and analyze large amounts of data by creating interactive tables.Macros: Automated scripts that allow users to automate repetitive tasks within Excel.
