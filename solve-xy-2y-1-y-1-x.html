1. solve the given differential equations by using appropriate substitution.(x²+2y²)dx/dy=xy, y(-1)=1​


Answer:

Given that the differential equation is

[tex](x^2+2y^2)\dfrac{dx}{dy}=xy[/tex]

can be written in the form of

[tex]\dfrac{dy}{dx}=\dfrac{x^2+2y^2}{xy}=\dfrac{x}{y}+\dfrac{2y}{x}[/tex]

Let [tex]y=vx[/tex], which implies that

[tex]\dfrac{dy}{dx}=v+x\dfrac{dv}{dx}[/tex]

Substituting these values, we get

[tex]v+x\dfrac{dv}{dx}=\dfrac{1}{v}+2v \\\\x\dfrac{dv}{dx}=\dfrac{1}{v}+v \\\\x\dfrac{dv}{dx}=\dfrac{1+v^2}{v} \\\\\dfrac{v}{1+v^2} dv=\dfrac{1}{x} dx[/tex]

Integrating on both sides, we get

[tex]\displaystyle\int\dfrac{v}{1+v^2}\:dv=\displaystyle\int \frac{1}{x}\:dx\\\\\frac{\ln\left| v^2+1 \right|}{2}=\ln\left|x \right|+c\\\\\ln\left| v^2+1 \right|=2\ln\left|x \right|+c\\\ln\left| v^2+1 \right|=\ln\left|x^2 \right|+c\\\ln\left| v^2+1 \right|-\ln\left|x^2 \right|=c\\\\\ln\left| \dfrac{v^2+1}{x^2} \right|=c\\\\e^{\ln\left| \dfrac{v^2+1}{x^2} \right| }=e^c\\\\\frac{v^2+1}{x^2}=c\\\\ v^2+1=cx^2\\v^2=-1+cx^2\\\left(\dfrac{y}{x}\right)^2 =-1+cx^2\\\\y^2=x^2(-1+cx^2)[/tex]

In terms of y,

[tex]y=x\sqrt{-1+cx^2} \:\text{ or }\: y=-x\sqrt{-1+cx^2}[/tex]

Given the initial condition [tex]y(-1)=1[/tex], we solve for the value of c:

[tex]y^2=x^2(-1+cx^2)\\1^2=(-1)^2\:[-1+c(-1)^2]\\c=2[/tex]

Therefore,

[tex]y=x\sqrt{-1+2x^2}\:\text{ or }\: y=-x\sqrt{-1+2x^2}\quad\leftarrow\text{ANSWER}[/tex]

But you can also show the final answer as one:

[tex]y^2=x^2(-1+2x^2)}\quad\leftarrow\text{ANSWER}[/tex]

30. Find all x and y such that x and y are real numbers, xy(xy+1)+(xy+3)-2(x+y+1/2)-(x+y-1)2=0


Answer:

(x,y) = (±1,±1)

Step-by-step explanation:

Let us try to factor the left-hand side of the equation:

xy(xy+1) + (xy+3) - 2(x+y+1/2) - (x+y-1)²

Let a = xy, b = x + y, the left-hand side becomes

a(a+1) + (a+3) - 2(b+1/2) - (b-1)²

= a² + a + (a+3) - 2b - 1 - b² + 2b - 1

= a² + (a+a) + (-2b+2b) + (3-1-1) - b²

= a² + 2a + 1 - b²

Notice a² + 2a + 1 is a perfect square trinomial, because:

The first term a² is the square of a: (a)² = a²The last term 1 is the square of 1: (1)² = 1The middle term is twice the product of a and 1: 2a = 2(a)(1)

By inspection, a² + 2a + 1 = (a + 1)², it follows

a² + 2a + 1 - b² = (a + 1)² - b²

Use the difference of squares formula: a² - b²=(a + b)(a - b)

(a + 1)² - b² = (a + 1 + b)(a + 1 - b)

Put xy and x + y back, the left-hand side becomes

(xy + 1 + x + y)(xy + 1 - x - y)

= (xy + x + y + 1)[xy - x - (y - 1)]

Factor out x:

[x(y + 1) + y + 1][x(y - 1) - (y - 1)]

= (x + 1)(y + 1)(x - 1)(y - 1)

We know that this is equal to zero, hence:

x + 1 = 0, x = -1y + 1 = 0, y = -1x - 1 = 0, x = 1y - 1 = 0, y = 1

Therefore, the solution set is (x,y) = (±1,±1)
