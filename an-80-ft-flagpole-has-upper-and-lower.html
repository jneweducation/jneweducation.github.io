11. On level ground, the base of a tree is 20 ft from the bottom of a 42 ft flagpole. The tree is shorter than the pole. At a certain time, their shadows end at the same point 60 ft from the base of the flagpole. How tall is the tree?


Answer:

We can solve this problem using similar triangles and proportions. Let's call the height of the tree "h".

First, we can use the fact that the flagpole is 42 ft tall and its shadow reaches 60 ft from the base to find the length of the flagpole's shadow. Since the shadow is proportional to the height of the object, we can set up a proportion:

(flagpole height)/(flagpole shadow length) = (tree height)/(tree shadow length)

Step-by-step explanation:

Plugging in the values we know, we get:

42/x = h/(x - 20)

where x is the length of the flagpole's shadow (and also the length of the tree's shadow).

Solving for x, we get:

x = 60

Substituting that value back into the proportion and solving for h, we get:

42/40 = h/40

h = 42*40/60

h = 28

Therefore, the height of the tree is 28 ft.
