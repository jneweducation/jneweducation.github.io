What is Python programming language?

Python is an interpreted high-level general-purpose programming language. Its design concept increases code readability by using a lot of indentation. Its language features and object-oriented approach are designed to help programmers write clear, logical code for both small and large-scale projects.

Python is dynamically typed and garbage-collected. Structured (particularly procedural) programming, object-oriented programming, and functional programming are among the programming paradigms it supports. It is commonly described to as a "batteries included" language due to its vast standard library.

Python was created by Guido van Rossum in the late 1980s as a successor to the ABC programming language, and it was originally released in 1991 as Python 0.9.0. Python 2.0 was released in 2000, with additional capabilities such as list comprehensions and a garbage collection mechanism that detects cycles (in addition to reference counting). Python 3.0 was issued in 2008, and it was a substantial difference of the language that was not backwards compatible. Python 2 was deprecated in 2020, with version 2.7.18.

26. python create programming with 13 subject computing grades if its passing it say: Congratulations if not passing its say:Better luck next time​


Answer:

# initialize variables

total = 0

passing_grade = 60

num_subjects = 13

# get input from user for each subject

for i in range(num_subjects):

   grade = int(input("Enter grade for subject " + str(i+1) + ": "))

   total += grade

# calculate average grade

average = total / num_subjects

# check if passing or failing

if average >= passing_grade:

   print("Congratulations! Your average grade is", average)

else:

   print("Better luck next time. Your average grade is", average)

EXPLANATION

In this program, we use a for loop to ask the user to enter the grade for each subject. We keep track of the total grade by adding each grade to the total variable. After all the grades have been entered, we calculate the average grade by dividing the total by the number of subjects.

Finally, we use an if statement to check if the average grade is greater than or equal to the passing_grade (which we set to 60 in this example). If it is, we print a congratulatory message along with the average grade. If it is not, we print an encouraging message along with the average grade.
