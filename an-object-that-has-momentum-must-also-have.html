2. If an object has kinetic energy, then it must also have a. impulse b. momentum c. torque d. acceleration


The answer is impulse

Because,
[tex]KE= \frac{1}{2} mv^2[/tex]
[tex]\because F=ma \to m= \frac{F}{a} [/tex]
[tex]\therefore KE= \frac{1}{2} Fav^2[/tex]

Where,
m is mass
v is velocity
a is acceleration

27. an object has a momentum of 85 kg.m/s and hits the stationary object making the second starts to move. If the first object ends with a momentum of 25 kg.m/s, What is the momentum of the second object.


Answer:

Total momentum = 85 kg.m/s + 0 (since the second object is stationary)

Total momentum = 85 kg.m/s

After the collision:

Total momentum = momentum of first object + momentum of second object

Total momentum = 25 kg.m/s + p2

Total momentum = 25 kg.m/s + p2

Since the total momentum is conserved, we can set the two expressions for total momentum equal to each other:

85 kg.m/s = 25 kg.m/s + p2

p2 = 85 kg.m/s - 25 kg.m/s

p2 = 60 kg.m/s

Therefore, the momentum of the second object is 60 kg.m/s
