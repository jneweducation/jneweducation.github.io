8. prove (1-cot^2x/1+cot^2x) = sin^2x - cos^2x​


We can start with the left-hand side of the equation and use the identity for cotangent:

(1 - cot^2x) / (1 + cot^2x)

= [(1 - cos^2x/sin^2x) / (1 + cos^2x/sin^2x)] (since cot^2x = cos^2x/sin^2x)

= [(sin^2x - cos^2x) / sin^2x * (1 + cos^2x/sin^2x)] (multiplying numerator and denominator by sin^2x)

= [(sin^2x - cos^2x) / (sin^2x + cos^2x)] * [(sin^2x + cos^2x) / sin^2x]

= (sin^2x - cos^2x) / sin^2x

= sin^2x / sin^2x - cos^2x / sin^2x

= 1 - cos^2x / sin^2x

= sin^2x - cos^2x

Therefore, we have shown that (1 - cot^2x) / (1 + cot^2x) = sin^2x - cos^2x.

Answer:

1.) The LCD of 1/2x-1 - 1/2x-1 is simply 2x-2.

2.) To find the LCD of 2x/2x-1 + 1/3x-1, we need to factor the denominators. 2x-1 and 3x-3 both have a common factor of (x-1). So the LCD is (x-1)(2x-1)(3).

3.) The LCD of 2x+1/2 - 1/2x-1 is (2)(2x-1).

4.) To find the LCD of x+1/x-1 + x-1/x+1, we need to factor the denominators. (x-1) and (x+1) are already in factored form. So the LCD is (x-1)(x+1).

5.) The LCD of 2x+1/2x-1 - 1/2x-1 is simply 2x-1.
