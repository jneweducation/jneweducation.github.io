15. Fill in the gaps in this arithmetic sequence: -3, ___, ___, ___, ___, 17


Add 4 each time:
-3, 1, 5, 9, 13, 17Arithmetic Sequence

Given:
[tex] a_{1} [/tex] = -3
[tex] a_{5} [/tex] = 17

We can solve for the common difference using the formula:
d = [tex] \frac{a_m - a_k}{m - k} [/tex]   [in which m is the higher term]
d = [tex] \frac{17 - (-3)}{5 - 1} [/tex]
d = [tex] \frac{20}{4} [/tex]
d = 5

To fill in the gaps, we may use the formula:
[tex] a_{n} [/tex] = [tex] a_{1} [/tex] + (n-1) d

[tex] a_{2} [/tex] = [tex] a_{1} [/tex] + (2-1) d
[tex] a_{2} [/tex] = -3 + (1) 5
[tex] a_{2} [/tex] = -3 + 5
[tex] a_{2} [/tex] = 2

[tex] a_{3} [/tex] = [tex] a_{1} [/tex] + (3-1) d
[tex] a_{3} [/tex] = -3 + (2) 5
[tex] a_{3} [/tex] = -3 + 10
[tex] a_{3} [/tex] = 7

[tex] a_{4} [/tex] = [tex] a_{1} [/tex] + (4-1) d
[tex] a_{4} [/tex] = -3 + (3) 5
[tex] a_{4} [/tex] = -3 + 15
[tex] a_{4} [/tex] = 12

Therefore, the arithmetic sequence will be:
-3, 2, 7, 12, 17

28. fill in the gaps in this arithmetic sequence: -3,_,_,_,_,_,12


Formula in finding terms in an arithmetic progression is

An = A1 +(n-1)d

--------------------------

You are given that:

An = A7 = 12

A1 = -3

n = 7

-------------------------

get the value of 'd' or the common difference to find the other terms

An = A1 + (n-1) d

12 = -3 + (7-1) d

12 = -3 + 6d

12 + 3 = 6d

15 = 6d

d = 15/6

------------------------

Using d you'll have the other terms

A2 = A1 + (n-1)d

A2 = -3 + (2-1)15/6

A2 = -3 + 15/6

A2 = -1/2

----------------------

A3 = A1 + (n-1)d

A3 = -3 + (3-1) 15/6

A3 = 2

--------------------

A4 = 9/2
