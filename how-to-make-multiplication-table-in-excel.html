1. How do you sum multiple rows in Excel?


Answer:

If you need to sum a column or row of numbers, let Excel do the math for you. Select a cell next to the numbers you want to sum, click AutoSum on the Home tab, press Enter, and you're done. When you click AutoSum, Excel automatically enters a formula (that uses the SUM function) to sum the numbers.

19. So in Microsoft excel, say I have to digitalize the class funds for example, how do I make excel put a negative number towards the days they didn't pay 5 pesos? Like I could just leave a blank in the table and excel will put a negative 5 towards the cell I selected to put the negative number into.


Answer:

try this

In Microsoft Excel, you can use a simple formula to insert a negative number into a cell based on certain conditions. For example, if you want to insert a negative number into a cell if it is blank, you can use the following formula:

Copy code

=IF(ISBLANK(A1),-5,A1)

In this formula, A1 is the cell where you want to insert the negative number. The ISBLANK function checks if the cell is blank, and if it is, the formula returns the number -5. Otherwise, it returns the value of the cell.

To use this formula, simply select the cell where you want to insert the negative number and type the formula into the formula bar. Then, press Enter to apply the formula. The cell will now show a negative number if it was previously blank, or the original value if it was not blank.

Note: This formula only works if the cell is truly blank, meaning it has no value and no formula. If the cell has a formula that returns an empty string or a zero value, this formula will not work. In that case, you would need to use a different formula that checks for those specific conditions.
