22. circumference of the circle 68mRadius:diameter:circumference:(need po ng formula sa circumference)​


Answer:

The formula for the circumference of a circle is:

C = 2πr

where C is the circumference, π (pi) is a mathematical constant approximately equal to 3.14, and r is the radius of the circle.

Given that the circumference of the circle is 68 meters, we can use the formula to find the radius and diameter.

First, we can solve for the radius:

C = 2πr

68 = 2πr

r = 68/(2π)

r ≈ 10.82 meters

The radius of the circle is approximately 10.82 meters.

Next, we can solve for the diameter:

d = 2r

d = 2(10.82)

d ≈ 21.64 meters

The diameter of the circle is approximately 21.64 meters.

Finally, we can use the formula to find the circumference directly using the radius we already calculated:

C = 2πr

C = 2π(10.82)

C ≈ 68.03 meters

Therefore, the circumference of the circle is approximately 68.03 meters.
