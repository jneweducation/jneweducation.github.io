9. If PC=3, AC=5x+1, PD = 4, and PB = 4x+2, find PA and PB.​


✒️POWER THEOREM

[tex]••••••••••••••••••••••••••••••••••••••••••••••••••[/tex]

[tex] \large\underline{\mathbb{ANSWER}:} [/tex]

[tex] \qquad\Large\rm \:\: PA = 24 \: units [/tex]

[tex] \qquad\Large\rm \:\: PB = 18 \: units [/tex]

[tex]••••••••••••••••••••••••••••••••••••••••••••••••••[/tex]

[tex] \large\underline{\mathbb{SOLUTION}:} [/tex]

» Solve for x by applying the Secant-Secant Power Theorem.

[tex] (PA)(PC) = (PB)(PD) [/tex]

[tex] (AC + PC)(PC) = (DB + PD)(PD) [/tex]

[tex] (5x + 4)(3) = (4x + 2)(4) [/tex]

[tex] 15x + 12 = 16x + 8 [/tex]

[tex] 15x - 16x = 8 - 12[/tex]

[tex] \text-x = \text-4[/tex]

[tex] x = 4[/tex]

» Thus, the value of x is 4. Substitute x to the segment PB and segment PA.

Segment PB:

[tex] PB = 4x + 2 [/tex]

[tex] PB = 4(4) + 2 [/tex]

[tex] PB = 16 + 2 [/tex]

[tex] PB = 18 [/tex]

Segment PA:

[tex] PA = 5x + 4 [/tex]

[tex] PA = 5(4) + 4 [/tex]

[tex] PA = 20 + 4 [/tex]

[tex] PA = 24 [/tex]

[tex] \therefore [/tex] Segment PB is 18 units while Segment PA is 24 units.

29. If PC=3, AC=5x+1, PD = 4, and PB = 4x+2, find PA and PB.​​


Answer:

PA=5 and PB=3.2.

Step-by-step explanation:

We can solve for PA and PB using the information that we have.

First, to find PA, we can use the line equation AC=5x+1 and PC=3. Since AC and PC are on the same line, they have the same slope and y-intercept.

Therefore, PC=3, AC=5x+1, PD = 4, and PB = 4x+2, find PA and PB. Give a detailed

solution

We can solve for PA and PB using the information that we have.

First, to find PA, we can use the line equation AC=5x+1 and PC=3. Since AC and PC are on the same line, they have the same slope and y-intercept.

Therefore, we can set AC=PC and solve for x.

5x+1=3

5x=2

x=2/5

We can now plug in x=2/5 into the equation AC=5x+1 to find AC when x=2/5.

AC=5(2/5)+1

AC=4

Now that we know AC=4, we can use this information to find PA.

We can use the equation PA=2AC-PC to solve for PA.

PA=2(4)-3

PA=5

We can now solve for PB using the equation PB=4x+2 and the known value of x=2/5.

PB=4(2/5)+2

PB=3.2

Therefore, PA=5 and PB=3.2.
