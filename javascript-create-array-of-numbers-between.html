1. Finding the average of three numbers mathematical operations in JavaScript arrays. ​


Answer:

To find the average of three numbers in a JavaScript array, you could use the following code:

Copy code

// Define the array of numbers

var numbers = [4, 8, 12];

// Calculate the sum of the numbers

var sum = numbers.reduce(function(a, b) {

 return a + b;

}, 0);

// Calculate the average of the numbers

var avg = sum / numbers.length;

// Print the average

console.log(avg); // 8

Explanation:

This code uses the reduce() method to calculate the sum of the numbers in the array, and then divides that sum by the number of elements in the array to find the average. The reduce() method iterates over the elements of the array, applying a specified function to each element to reduce the array to a single value. In this case, the function simply adds the current element to the running total.

18. Create a program in c++ that will accept a length of an array and accecpt numbers based on the size of an array, and it will only accept numbers from -100 to 100. After accepting the numbers,the program will ask the user of an opereator to calculate the numbers.​


Answer:

Explanation:

#include <iostream>

using namespace std;

int main() {

  int n, i;

  cout << "Enter the length of the array: ";

  cin >> n;

  int arr[n];

 

  // Accept numbers from user and validate them

  cout << "Enter " << n << " numbers between -100 to 100:\n";

  for (i = 0; i < n; i++) {

     cin >> arr[i];

     if (arr[i] < -100 || arr[i] > 100) {

        cout << "Invalid input! Number must be between -100 to 100.\n";

        i--; // repeat this iteration

     }

  }

 

  // Ask user for operator

  char op;

  cout << "Enter an operator (+, -, *, /): ";

  cin >> op;

 

  // Calculate the result based on operator

  int result = arr[0];

  for (i = 1; i < n; i++) {

     switch(op) {

        case '+':

           result += arr[i];

           break;

        case '-':

           result -= arr[i];

           break;

        case '*':

           result *= arr[i];

           break;

        case '/':

           if (arr[i] == 0) {

              cout << "Cannot divide by zero!\n";

              return 0;

           }

           result /= arr[i];

           break;

        default:

           cout << "Invalid operator!\n";

           return 0;

     }

  }

 

  // Output the result

  cout << "The result is " << result << endl;

  return 0;

}

22. Give a definition and an example syntax for each topic: 1. Declaration of Array 2. Initializing Array 3. Assigning an Array Element 4. Accessing an Array Element 5. Using the Array Length 6. Sorting elements of an Array 7. Creating Single Dimensional Array​


Answer:

Declaration of Array: This is the process of defining the data type of an array, as well as the name of the array that will be used to refer to it in the program.

Syntax: dataType[] arrayName;

Example: int[] numbers;

Initializing Array: This is the process of setting the initial values for an array at the time of its creation.

Syntax: dataType[] arrayName = {value1, value2, value3, ...};

Example: int[] numbers = {1, 2, 3, 4, 5};

Assigning an Array Element: This is the process of setting the value of a specific element in an array.

Syntax: arrayName[index] = value;

Example: numbers[0] = 10;

Accessing an Array Element: This is the process of retrieving the value of a specific element in an array.

Syntax: arrayName[index];

Example: int x = numbers[2];

Using the Array Length: This is a property that returns the number of elements in an array.

Syntax: arrayName.length;

Example: int length = numbers.length;

Sorting elements of an Array: This is the process of arranging the elements of an array in ascending or descending order.

Syntax: Arrays.sort(arrayName);

Example: Arrays.sort(numbers);

Creating Single Dimensional Array: This is an array that contains a list of values of the same data type.

Syntax: dataType[] arrayName = new dataType[arraySize];

Example: int[] numbers = new int[5];
