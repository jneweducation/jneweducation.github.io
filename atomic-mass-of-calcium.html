Molar mass of Calcium = 40.08 g/mol

That is to say, 1 mol of Calcium has a mass of 40.08 g

Also, according to Avogadro’s Number, 1 mole of Calcium has 6.022 x 10^23 atoms.

Therefore, if you have 1.204 x 10²⁴ calcium atoms, this must have a mass of {(1.204 x 10²⁴)/(6.022 x 10²³)} x 40.08 g = 80.13 g

27. Difference between a calcium atom and a calcium ion


The difference between a calcium atom and a calcium ion is the number of electrons

Electron atom Ca = 20 while calcium ion (Ca²⁺)= 18

Further explanation

The Atomic Number (Z) indicates the number of protons in an atom of an element. If the atom is neutral then the number of protons will be equal to the number of electrons. So the atomic number can also indicate the number of electrons.

 So atomic number = number of protons = number of electrons

Mass Number (A) is the sum of protons and neutrons

Mass Number (A) = Number of protons + Number of Neutrons

So that the relationship between atomic numbers and mass numbers can be formulated as follows:

Atomic Number (Z) = Mass Number (A) - Number of Neutrons

In the following element notation,

[tex]\large {{{A} \atop {Z}} \right X}[/tex]

X = symbol of elemental atom

A = mass number

   = number of protons + number of neutrons

Z = atomic number

   = number of protons = number of electrons, on neutral elements

The Ca element is in groups 2A and period 4 in the periodic system of elements with atomic number 20 and mass number 40

So the composition:

₂₀⁴⁰Ca,

proton = no.atom = 20

electron = proton = 20

neutron = no. mass - proton = 40 - 20 = 20

Ca²⁺

While Ca ions are generally in the form of Ca²⁺ (cations)

This cation is obtained by removing 2 electrons, with the aim of getting stability like the configuration of the noble gas electron ₁₈Ar

So the composition:

proton = no.atom = 20

electron = 18 (release 2 electrons) -> number of protons - charge

neutron = no. mass - proton = 40 - 20 = 20
