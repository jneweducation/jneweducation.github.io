13. 4. Find the 10 term of the arithmetic sequence 17. 13. 9.A. -59B. -55C. -51D.53​


ARITHMETIC SEQUENCE

Find the 10th term of the arithmetic sequence 17, 13, 9...

A. -59

B. -55

C. -51

D. 53

First, find the common difference.

[tex]a2 - \green{a1} = \purple d \\ a3 - a2 = \purple d \\ \\ 13 - \green{17} = \purple {-4} \\ 9 - 3 = \purple { - 4}[/tex]

Find the missing term by using the formula, subtitute the numbers. (a1) is the first term, (d) is the difference, and (n) for the place of the term we were finding.

[tex] \green {a1} = \green{17} \\ \purple d = \purple{ - 4} \\ \blue n = \blue{10}[/tex]

[tex]a \blue {n} = \green{a1} + ( \blue{n} - 1) \purple {d} \\ \\ a \blue {10} = \green{17} + ( \blue{10} - 1) (\purple { - 4}) \\ a \blue {10} = \green{17} + \blue{9} \: (\purple { - 4}) \\ a \blue {10} = \green{17} + (\blue{ - 36}) \\ \boxed{a \blue {10} = \blue{ - 19}}[/tex]

Thus, the answer is none of the choices. The 10th term of the given sequence is -19

24. 3. I am thinking of a number. Thrice my number plus 17 equals 59. What is the number?​


Answer:

The number is 14.

Step-by-step explanation:

Let n be the number that when multiplied by 3 and added by 17, the result is 59. So if we want to find the number, we will make an equation to solve for n.

[tex] \sf 3n + 17 = 59 [/tex]

[tex] \sf 3n = 59 - 17 [/tex]

[tex] \sf 3n = 42 [/tex]

[tex] \sf \frac{\cancel{3}n}{\cancel{3}} = \frac{42}{3} [/tex]

[tex] \large \sf n = \green{14} [/tex]

Therefore, the number is 14. So if we multiply this value by three and add the product by 17, the result is 59.

Check:

[tex] \sf 3n + 17 = 59 [/tex]

[tex] \sf 3(14) + 17 \overset{?}{=} 59 [/tex]

[tex] \sf 42 + 17 \overset{?}{=} 59 [/tex]

[tex] \sf 59 \overset{\checkmark}{=} 59 [/tex]
