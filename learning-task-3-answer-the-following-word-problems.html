Learning Task 3:Answers:4 flowers14 boys21 kilometersSolutions:

1. Given: 10 flowers

              [tex]\frac{3}{5}[/tex] - yellow flowers

Asked: The number of flowers which are not yellow.

Number Sentence: 10 - [tex]\frac{3}{5}[/tex](10) = N

Answer:  10 - [tex]\frac{3}{5}[/tex](10) = N

10 - [tex]\frac{30}{5}[/tex] = N

10 - 6 = N

4 = N

Therefore, 4 flowers are not yellow.

2. Given: 7 pupils - number of pupils per group

               [tex]\frac{2}{7}[/tex] - boys among the class

Asked: The number of boys in the class.

Number Sentence: 49 - [tex]\frac{5}{7}[/tex](49) =N

49 - [tex]\frac{245}{7}[/tex] = N

49 - 35 = N

14 = N

Therefore, there 14 boys in the class.

3. Given: 1 [tex]\frac{1}{2}[/tex] kilometer - distance between every billboards

               14 - number of billboards

Asked: The distance covered by the billboards.

Number Sentence: 14 × 1 [tex]\frac{1}{2}[/tex] = N

Answer: 14 × [tex]\frac{3}{2}[/tex] = N

[tex]\frac{42}{2}[/tex] = N

21 = N

Therefore, the billboards covered 21 kilometers.

Wednesday

Asked: The number of hours spent for Math

homework from Monday to Wednesday.

Number Sentence: ⅔ + ¾ + ⅖ = N

Answer: ⅔ + ¾ + ⅖ = N

Using the least common denominator of 3, 4, and 5 express the fractions as similar.

(20)(2)/60 + (15)(3)/60 + (12)(2)/60 = N

40/60 + 45/60 + 24/60 = N

85/60 + 24/60 = N

109/60 = N

1 49/60 = N

Therefore, Jose completed his Math homework for 1 hours.

solution:

2 ⅝ cm of wire needed for the project in TLE

Asked: The length of wire that Mika needs to cut.

Answer: 3 7/16 - 2 ⅝ = N

Rename the fraction as similar fractions.

7/16 = 7/16

5/8 = 10/16

3 7/16 - 2 10/16 = N

since is greater than we need to rename again the fraction.

Number Sentence: 4 ½ ÷ 6 = N

Change 4 ½ to improper fraction.

4 ½ = 9/2
