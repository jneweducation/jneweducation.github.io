10. difference between mysql and nosql​


Differences between MySQL and NoSQL are:

1. Data structure: MySQL uses a fixed schema to define the structure of the data, while NoSQL databases allow for flexible schema.

2. Scalability: NoSQL databases are designed to handle large amounts of data and can scale horizontally by adding more servers, while MySQL can scale vertically by adding more resources to a single server.

3. Querying: MySQL uses SQL for querying data, while NoSQL databases use different models and query languages depending on the type of database.

4. Data consistency: MySQL provides strong data consistency, while NoSQL databases provide eventual consistency, which means that the data may not be immediately consistent across all nodes in the database.

5. Data types: MySQL supports a wide range of data types, while NoSQL databases may have limited support for certain data types.
