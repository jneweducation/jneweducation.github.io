1. A forklift in a warehouse lifts 1 000n of boxes, 34m from the floor.compute the work done by the forklift


[tex]\tt{\huge{\blue{Explanation:}}}[/tex]

The work done by an object exerting a force F at a distance d is given by

[tex]\boxed{W = Fd}[/tex]

where:

W = work

F = force

d = distance

[tex]\tt{\huge{\red{Solution:}}}[/tex]

Solving for the work

W = Fd

W = (1,000 N)(34 m)

[tex]\boxed{W = \text{34,000 J}}[/tex]

Therefore, the work done by the forklift is 34,000 J.

A forklift in a warehouse lifts 1000 N of boxes, 34 M from the floor. Compute the work done by the forklift.

Letter A. 34000 J

the work done by the forklift is equal to 34,000 J or 34,000 kg·m²/s²

Note:

the formula of work is ( W = F × d ).

Given Data:

F ( Force ) = 1000 N or 1000 kg·m/s²

d ( distance ) = 34 m

W ( Work ) = ? unknown

Solution:

We cam use the formula of work which is ( W = F × d ).

W = F × d

    = 1000 kg·m/s² × 34 m

    = 34,000 kg·m²/s²

    = 34,000  J
