5. 1.) Imput student name year level units enrolled Where: tuition fee = rate per unit * units enrolled Rate per unit table. year level rate per unit 100 1 2 output: student name tuition fee 125 3 150 4,5 200 create two programs, using if and switch​


Answer:

Here are two programs, one using if statements and the other using switch statements, to compute the tuition fee of a student based on their year level and number of units enrolled:

Program using if statements:

```

#include <iostream>

#include <string>

using namespace std;

int main() {

string name;

int year_level, units_enrolled;

float rate_per_unit, tuition_fee;

cout << "Enter student name: ";

getline(cin, name);

cout << "Enter year level: ";

cin >> year_level;

cout << "Enter units enrolled: ";

cin >> units_enrolled;

if (year_level == 1) {

rate_per_unit = 100;

} else if (year_level == 2) {

rate_per_unit = 125;

} else if (year_level == 3) {

rate_per_unit = 150;

} else if (year_level == 4 || year_level == 5) {

rate_per_unit = 200;

} else {

cout << "Invalid year level." << endl;

return 0;

}

tuition_fee = rate_per_unit * units_enrolled;

cout << "Tuition fee of " << name << " is " << tuition_fee << endl;

return 0;

}

```

Program using switch statements:

```

#include <iostream>

#include <string>

using namespace std;

int main() {

string name;

int year_level, units_enrolled;

float rate_per_unit, tuition_fee;

cout << "Enter student name: ";

getline(cin, name);

cout << "Enter year level: ";

cin >> year_level;

cout << "Enter units enrolled: ";

cin >> units_enrolled;

switch (year_level) {

case 1:

rate_per_unit = 100;

break;

case 2:

rate_per_unit = 125;

break;

case 3:

rate_per_unit = 150;

break;

case 4:

case 5:

rate_per_unit = 200;

break;

default:

cout << "Invalid year level." << endl;

return 0;

}

tuition_fee = rate_per_unit * units_enrolled;

cout << "Tuition fee of " << name << " is " << tuition_fee << endl;

return 0;

}

```

Both programs will prompt the user to enter the student's name, year level, and units enrolled, and then compute and display the tuition fee using the rate per unit table provided. If an invalid year level is entered, the program will display an error message and terminate.

13. The average tuition fee per semester in the collage level is at 50,000. if yearly tuition increase is expected to be at around 4% , how much will tuition cost 15 years from now?​


Answer:

Tuition will cost 80,000

Step-by-step explanation:

let n= tuition cost 15 years from now

n= 50,000(4%)(15) + 50,000

Each year tuition increases by 4%, which means in 15 years tuition increases by 60%

4 times 15=60

so we simply find 60% of 50,000 and add to the original value

50,000 times 0.60 = 30,000

n= 30,000+50,000

n= 80,000


14. Tuition fee is increased by 25% since five years. If the tuition fee then was ₱5 000, how much is the tuition fee now?


Answer: ₱6,250


Step-by-step explanation:

To get the answer,

First, you multiply it by 0.25.

It becomes:

5,000 × 0.25 = 1,250.

But, since the tuition is increased by 25%, we add 5000 to 1250.

5000 + 1250 = 6250.

Therefore, tuition today costs ₱6,250.
