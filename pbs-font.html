17. balance and identify the type of reactionpb +H³(90⁴) --> H² + pb³(PO⁴)​


Answer:

The given chemical equation represents a redox reaction, in which there is an exchange of electrons between the reactants.

Pb (lead) is oxidized from its zero oxidation state to +4 oxidation state, while H3(90⁴) (sulfuric acid) is reduced from +6 oxidation state to +4 oxidation state.

The balanced chemical equation for the reaction is:

Pb + 2H3(90⁴) → H2 + Pb3(PO4)2

In this equation, one atom of Pb reacts with two molecules of H3(90⁴) to form one molecule of H2 gas and one molecule of Pb3(PO4)2, which is a white precipitate.

Therefore, the type of reaction is a redox reaction, specifically an oxidation-reduction reaction.

25. If PC=3, AC=5x+1, PD = 4, and PB = 4x+2, find PA and PB.​​


Answer:

PA=5 and PB=3.2.

Step-by-step explanation:

We can solve for PA and PB using the information that we have.

First, to find PA, we can use the line equation AC=5x+1 and PC=3. Since AC and PC are on the same line, they have the same slope and y-intercept.

Therefore, PC=3, AC=5x+1, PD = 4, and PB = 4x+2, find PA and PB. Give a detailed

solution

We can solve for PA and PB using the information that we have.

First, to find PA, we can use the line equation AC=5x+1 and PC=3. Since AC and PC are on the same line, they have the same slope and y-intercept.

Therefore, we can set AC=PC and solve for x.

5x+1=3

5x=2

x=2/5

We can now plug in x=2/5 into the equation AC=5x+1 to find AC when x=2/5.

AC=5(2/5)+1

AC=4

Now that we know AC=4, we can use this information to find PA.

We can use the equation PA=2AC-PC to solve for PA.

PA=2(4)-3

PA=5

We can now solve for PB using the equation PB=4x+2 and the known value of x=2/5.

PB=4(2/5)+2

PB=3.2

Therefore, PA=5 and PB=3.2.
