7. QuizDirection: Prove the given statement using direct proof and indirect proof.if 1/5 (x + 10) = 3. then x = 5Prove: 1/5 (x+10) = 3 Given: x = 5​


Answer:

Direct Proof:

To prove 1/5 (x+10) = 3, we substitute x=5 into the equation and simplify:

1/5 (x+10) = 1/5 (5+10) = 1/5 (15) = 3

Therefore, 1/5 (x+10) = 3 is true when x = 5.

Indirect Proof:

Assume that 1/5 (x+10) ≠ 3 when x = 5. This means that:

1/5 (5+10) ≠ 3

Simplifying the left-hand side, we get:

1/5 (15) ≠ 3

3 ≠ 3

This is a contradiction, as 3 is equal to 3. Therefore, our assumption that 1/5 (x+10) ≠ 3 when x = 5 is false. Hence, 1/5 (x+10) = 3 is true when x = 5.

15. Activity 1. Do the following: A. Prove using Direct Proof Given: Odd integer m and even integer n (m = 2k +1 and n = 2k) Prove: mn is even number (mn = 2r) B. Prove using Indirect Proof If x= 3, then 3x+5=10​


A. Direct Proof:

Given: Odd integer m and even integer n (m = 2k +1 and n = 2k)

Prove: mn is even number (mn = 2r)

Step 1: Assume m = 2k +1 and n = 2k, where k is an integer.

Step 2: Multiply m and n: mn = (2k +1) * (2k) = 4k^2 + 2k

Step 3: 4k^2 + 2k can be expressed as 2 * (2k^2 + k), which is in the form of 2r, where r is an integer.

Step 4: Therefore, mn is an even number.

B. Indirect Proof:

If x= 3, then 3x+5=10

Assume x is not equal to 3. Then we have 3x+5 ≠ 10

We can substitute x=2, x=4, or any other value that is not equal to 3 and we will find that 3x+5 ≠ 10.

Since our assumption that x is not equal to 3 leads to a contradiction, it must be false. Therefore, x must equal 3. And thus 3x+5=10
