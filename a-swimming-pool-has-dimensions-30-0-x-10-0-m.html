17. The length of the rectangular swimming pool is 2 meters less than 3 times the width. If the area is 65 m². Find the dimensions of the swimming pool.​


Answer:

The area of a rectangle is given by the formula length*width. We can use this to set up an equation using the information given in the problem.

Let x be the width of the pool

length = 3x - 2

Area = (3x - 2)x = 65

Expanding and solving the equation we get

3x^2 - 2x - 65 = 0

By factoring the equation we get

(3x+13)(x-5) = 0

which gives x = -13/3 or x = 5

Since the width of a pool cannot be negative, the width of the pool is 5 meters.

The length of the pool is 3x - 2 = 3(5) - 2 = 13 meters
