19. the length of a rectangular field is 8m longer than it's worth if the area is 2900 me find the dimensions of the lot​


Answer:

Width = 50m, Length = 58m

Step-by-step explanation:

Let x and x + 8 be the width and length respectively

x(x + 8) = 2900

Distributive Property

x^2 + 8 = 2900

Transpose 2900 and change its sign

x^2 + 8 - 2900 = 0

Factor the left side of the equation

(x - 50)(x + 58) = 0

Set all factors equal to 0

x - 50 = 0 or x + 58 = 0

x = 50 or x = -58

The width is 50m.

x + 8

50 + 8

= 58

The length is 58m.

Checking:

x (x + 8) = 2900

50 (50 + 8) = 2900

50 (58) = 2900

2900 = 2900 ☑
