16. 9. The dimensions of a rectangle with an area of 56 meters are (3x - 1) meters by(x +4) meters. Which of the following quadratic equation in standard formrepresents the situation? (3x-1)(x+4)A. 3x2 + 11x-60 = 0C. 3x2 + 11x - 52 = 0B. 3x2 + 11x + 60 = 0D. 3x2 + 11x + 52 = 0​


Answer:

A. 3x² + 11x - 60 = 0

Step-by-step explanation:

Area = length x width

or length x width = Area

(3x-1)(x+4) = 56

3x² + 12x - x - 4 = 56

3x² + 11x - 4 - 56 = 0

3x² + 11x - 60 = 0
