14. Activity 1.3 Finding the Unknown! A Complete the table below by finding the unknown. Principal Time 73.300 110.000.000 Rate 0.025% (b) 0.5% (3) B. After finding the unknown of the data above, compu for the Interest 15.000.​


PA BRAINLIEST!

Narito ang table na may kasama ang mga hinahanap na bilang:

| | Principal | Time | Rate | Interest |

|----------|-----------|-------|---------|----------|

| Value | 73,300 | 110.0 | 0.025% | 15,000 |

| Unknown 1 | | b | 0.5% | |

| Unknown 2 | | 3 | | |

Para mahanap ang Unknown 1 (b), kailangan nating gamitin ang formula para sa simple interest:

Interest = (Principal x Rate x Time) / 100

Substituting the values given:

15,000 = (73,300 x b x 110) / 100

Solving for b:

b = 15,000 x 100 / (73,300 x 110)

b = 0.236% (rounded to three decimal places)

Kaya naman ang sagot sa Unknown 1 (b) ay 0.236%.

Para naman sa Unknown 2, kailangan nating gamitin ang formula muli:

Interest = (Principal x Rate x Time) / 100

Substituting the values given:

15,000 = (Unknown 2 x 0.025% x 3) / 100

Solving for Unknown 2:

Unknown 2 = 15,000 x 100 / (0.025% x 3)

Unknown 2 = 20,000,000

Kaya naman ang sagot sa Unknown 2 ay 20,000,000.
