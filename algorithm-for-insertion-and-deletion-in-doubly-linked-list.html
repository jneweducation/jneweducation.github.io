4. inserting and deleting rows and columns


Answer:

Inserting and deleting rows and columns in a spreadsheet can be done in different ways depending on the software you are using. Here are some general steps you can follow:

Inserting Rows:

Select the row below where you want to insert the new row.

Right-click on the selected row and click on "Insert" or "Insert Row".

Alternatively, you can go to the "Home" tab and click on the "Insert" dropdown menu and select "Insert Sheet Rows".

Deleting Rows:

Select the row(s) you want to delete.

Right-click on the selected row and click on "Delete" or "Delete Row".

Alternatively, you can go to the "Home" tab and click on the "Delete" dropdown menu and select "Delete Sheet Rows".

Inserting Columns:

Select the column to the right of where you want to insert the new column.

Right-click on the selected column and click on "Insert" or "Insert Column".

Alternatively, you can go to the "Home" tab and click on the "Insert" dropdown menu and select "Insert Sheet Columns".

Deleting Columns:

Select the column(s) you want to delete.

Right-click on the selected column and click on "Delete" or "Delete Column".

Alternatively, you can go to the "Home" tab and click on the "Delete" dropdown menu and select "Delete Sheet Columns".

It is important to note that when you insert or delete rows or columns, it may affect the data in your spreadsheet. Therefore, it is recommended to save a backup copy of your data before making any changes.

7. it is a category of algorithms that is responsible for deleting an existing from a data structure​


Algorithm is a step-by-step procedure, which defines a set of instructions to be executed in a certain order to get the desired output. Algorithms are generally created independent of underlying languages, i.e. an algorithm can be implemented in more than one programming language.

From the data structure point of view, following are some important categories of algorithms −

Search − Algorithm to search an item in a data structure.

Sort − Algorithm to sort items in a certain order.

Insert − Algorithm to insert item in a data structure.

Update − Algorithm to update an existing item in a data structure.

Delete − Algorithm to delete an existing item from a data structure.

Characteristics of an Algorithm

Not all procedures can be called an algorithm. An algorithm should have the following characteristics −

Unambiguous − Algorithm should be clear and unambiguous. Each of its steps (or phases), and their inputs/outputs should be clear and must lead to only one meaning.

Input − An algorithm should have 0 or more well-defined inputs.

Output − An algorithm should have 1 or more well-defined outputs, and should match the desired output.

Finiteness − Algorithms must terminate after a finite number of steps.

Feasibility − Should be feasible with the available resources.

Independent − An algorithm should have step-by-step directions, which should be independent of any programming code.

29. what are the tools used in inserting and deleting comments?​


Answer:

How to Add a Comment

To add a comment in Microsoft Word, you first need to select the text you’re commenting on with the cursor. Once you’ve done that, you have three options available:

Go to the Review tab on the main ribbon and click New Comment

se the shortcut Ctrl + Alt + M (Windows) or Cmd + Alt + A (Mac)

Any of these will add a new comment in the place selected (it will appear in the right-hand margin). You can then click the comment and write whatever you need to say.

To remove a comment from a document, first select it and then either:

Go to Review > Comments and click Delete

Right-click the comment and select Delete Comment from the menu

And hey presto! The comment is gone. Alternatively, you can remove every comment in the document at once by going to Review > Comments and clicking on the little down arrow to open a new menu, then selecting Delete All Comments. Make sure you are ready to do this, though, as you might end up deleting an important comment by accident if you don’t check first!
