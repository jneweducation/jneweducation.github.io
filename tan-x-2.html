3. Prove the trigonometric identities :1 - tan²x ________ = 1 - 2 sin²x1 + tan²x​


Answer:

[tex] \frac{1 - {tan}^{2}x }{1 + {tan}^{2}x } = 1 - 2 {sin}^{2} x[/tex]

since:

[tex]1 + {tan}^{2} x = {sec}^{2} x[/tex]

[tex] \frac{1 - {tan}^{2} x}{ {sec}^{2}x } = 1 - 2 {sin}^{2} x[/tex]

[tex] \frac{1}{ {sec}^{2} x} - \frac{ {tan}^{2}x }{ {sec}^{2} x} = 1 - 2{sin}^{2} x[/tex]

[tex] {sec}^{2} x = \frac{1}{ {cos}^{2}x } [/tex]

[tex] {tan}^{2} x = \frac{ {sin}^{2}x }{ {cos}^{2}x } [/tex]

[tex] \frac{1}{ \frac{1}{ {cos}^{2} x} } - \frac{ \frac{ {sin}^{2} x}{ {cos}^{2} x} }{ \frac{1}{ {cos}^{2}x } } = 1 - 2 {sin}^{2} x[/tex]

arrange:

[tex] \frac{1}{1} \times \frac{ {cos}^{2} x}{1} = {cos}^{2} x[/tex]

[tex] \frac{ {sin}^{2}x }{ {cos}^{2} x} \times \frac{ {cos}^{2} x}{1} = {sin}^{2} x[/tex]

insert:

[tex] {cos}^{2} x - {sin}^{2} x = 1 - 2 {sin}^{2} x[/tex]

since:

[tex] {cos}^{2} x = 1 - {sin}^{2} x[/tex]

therefore:

[tex](1 - {sin}^{2} x) - {sin}^{2} x = 1 - 2 {sin}^{2} x[/tex]

[tex]1 - 2 {sin}^{2} x = 1 - 2 {sin}^{2} x[/tex]

13. y = arc tan x + arc tan 1\x​


Answer

y = arc tan x + arc tan 1\x

The expression y = arc tan x + arc tan 1/x is the sum of two inverse tangent functions. The inverse tangent function, also known as arctan or tan^-1, gives the angle whose tangent is a given value. In this expression, the first term, arc tan x, gives the angle whose tangent is x, and the second term, arc tan 1/x, gives the angle whose tangent is 1/x. Therefore, y is equal to the sum of these two angles.

It is not possible to solve for a specific value of y in this equation because the inverse tangent function is not a one-to-one function. The range of the inverse tangent function is (-π/2, π/2), which means that for a given value of y there can be multiple values of x that will satisfy the equation y = arc tan x + arc tan 1/x.

However, we can simplify the equation using the identity

arc tan (x) + arc tan (1/x) = (π/2) if x > 0

So, if x > 0 the solution of the equation y = arc tan x + arc tan 1/x is y = (π/2)
