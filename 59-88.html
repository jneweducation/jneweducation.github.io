1. what is the sum of -60 + (-59)+(-58)+...+87+88+89+90


Find n, the number of terms:

an = a₁ + (n-1)(d)
90 = -60 + (n-1)(1)
90 = -60 + n - 1
n = 90 + 61
n = 151

Solve for the sum of the sequence:

Sn = n/2 (a₁ + an)

S₁₅₁ = 151/2 (-60 + 90)

S₁₅₁ = 151/2 (30)

S₁₅₁ = 151 (15)

S₁₅₁ = 2,265   (Sum of the sequence)
