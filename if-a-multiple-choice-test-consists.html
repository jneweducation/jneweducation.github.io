11. Topic: ProbabilityA teacher gave a quiz consisting of 5 item matching test and a 5 item multiple choice with 4 choices each. If a student answer each question in the quiz, in how many ways can you answer the question?Please pasagot po ng tama​


Answer:

122,880

Step-by-step explanation:

For the first part, there are 5 items to match and 5 choices. We can choose the first item in 5 ways, then the second item in 4 ways (since one choice has already been used), then the third item in 3 ways, then the fourth item in 2 ways, and the last item in 1 way. So the total number of ways to match the items is:

5 x 4 x 3 x 2 x 1 = 120

For the second part, there are 5 multiple choice questions with 4 choices each. For each question, we can choose one of the 4 choices, so the total number of ways to answer the multiple choice questions is:

4 x 4 x 4 x 4 x 4 = 4^5 = 1024

To find the total number of ways to answer the entire quiz, we multiply the number of ways to answer the matching questions by the number of ways to answer the multiple choice questions:

120 x 1024 = 122,880

Therefore, there are 122,880 ways to answer the quiz.

13. Ms. Segovia gives a multiple choice test consisting of 15 questions with four choices each. There is only one correct answer for each question. If a student guesses the answer to each question, what is the probability that he gets 7 questions correctly?


Answer:

The probability of guessing the correct answer for one question on a multiple choice test with four choices is 1/4 or 25%.

To find the probability of getting 7 questions correct out of 15 by guessing, we can use the binomial probability formula:

P(k) = (n choose k) * p^k * (1-p)^(n-k)

where:

k = number of successful outcomes (getting 7 questions correct)

n = total number of trials (15 questions)

p = probability of success on one trial (1/4 or 25%)

So in this case:

P(7) = (15 choose 7) * (1/4)^7 * (3/4)^8

Using the binomial formula, we get:

P(7) = 6435 * (1/16384) * (9/16384) = 0.2028

So the probability of getting 7 questions correct out of 15 by guessing is approximately 20.28%.
