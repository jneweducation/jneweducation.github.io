1. Solve for he unknown P ( 8, r ) = 6720


Answer: r = 5


Step-by-step explanation:

Permutation formula:

P(n,r) = n! / (n - r)!


Solve for r:

6720 = 8! / (8 - r)!

Multiply both sides by (8 - r)!

6720 (8 - r)! = 8!

Divide both sides by 6780

(8 - r)! = 8! / 6720

Expand 8!

(8 - r)! = (1) (2) (3) (4) (5) (6) (7) (8) / 6720

(8 - r)! = 40320 / 6720

(8 - r)! = 6

Express 6 in factorial form, 6 = 3!

(8 - r)! = 3!

Since both sides have factorial sign, make (8 - r) = 3 as equality

8 - r = 3

Transpose

-r = -5

r = 5

11. A. 132

Solution:

9. To find the number of sets of questions that can be constructed, we use the combination formula:

n C r = n! / (r! * (n - r)!)

where n is the total number of questions (5) and r is the number of questions to be selected for a set (3).

Plugging in the values, we get:

5 C 3 = 5! / (3! * (5 - 3)!)

= 5! / (3! * 2!)

= (5 * 4 * 3 * 2 * 1) / [(3 * 2 * 1) * (2 * 1)]

= 10

Therefore, Ms. De Leon can construct 10 different sets of questions.

10. We are given P(9, r) = 3024. Using the formula for permutations:

P(n, r) = n! / (n - r)!

we can write:

9! / (9 - r)! = 3024

Multiplying both sides by (9 - r)!, we get:

9! = 3024 * (9 - r)!

Simplifying, we get:

9 * 8 * 7 * 6 * 5 * 4 * 3 * 2 * 1 = 3024 * (9 - r) * 8 * 7 * 6 * 5 * 4 * 3 * 2 * 1

Canceling out the common factors, we get:

9 - r = 6

Therefore, r = 3.

Hence, the answer is D. 6.

11. We need to find the number of ways 3 singers can be selected from a total of 12 singers, and then arrange them in order. This is a combination followed by a permutation.

The number of ways to select 3 singers from 12 is:

12 C 3 = 12! / (3! * (12 - 3)!)

= (12 * 11 * 10) / (3 * 2 * 1)

= 220

Once we have selected the 3 singers, we can arrange them in 3! = 6 ways.

Therefore, the total number of ways to arrange the first three singers is:

220 * 6 = 1,320

Hence, the answer is C. 1320.

12. We can simply use the formula for permutations:

P(n, r) = n! / (n - r)!

Plugging in the values, we get:

P(8, 5) = 8! / (8 - 5)!

= (8 * 7 * 6 * 5 * 4 * 3 * 2 * 1) / (3 * 2 * 1)

= 56 * 4 * 3 * 2

= 336

Therefore, P(8, 5) = 336.

13. We are given P(n, 4) = 5040. Using the formula for permutations:

P(n, r) = n! / (n - r)!

we can write:

n! / (n - 4)! = 5040

Multiplying both sides by (n - 4)!, we get:

n! = 5040 * (n - 4)!

Simplifying, we get:

n * (n - 1) * (n - 2) * (n - 3) = 5040

We can factor 5040 as:

5040 = 7 * 6 * 5 * 4 * 3 * 2 * 1

From this, we can see that n must be 12:

12 * 11 * 10 * 9 = 11880

11 * 10 * 9 * 8 = 7920

We can rule out n = 10 and n = 9 because neither of these expressions equals 5040.

Therefore, the answer is A. 12.

14. We know that:

x = P(n, n) = n!

y = P(n, n - 1) = n! / (n - 1)!

Dividing y by x, we get:

y / x = [n! / (n - 1)!] / n!

= 1 / n

Since n is greater than or equal to 2, we know that:

1 / n < 1

Therefore, y is less than x, or y < x. The answer is B. x > y.

[tex]\text{\# of cubes} = \text{Volume}_{\text{cuboid}} / \text{Volume}_{\text{cube}}\\\text{\# of cubes} = 6720/4^3\\\text{\# of cubes} = 6720/64\\\text{\# of cubes} = 105[/tex]

Therefore, 105 cubes of side 4 cm can be created from the cuboid.
