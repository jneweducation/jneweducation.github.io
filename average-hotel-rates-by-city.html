18. pls help, A hotel has 100 rooms and in the month of February, the hotel had an average daily rate (ADR) of PhP6,500 per room. The hotel had an occupancy rate of 80% for the month, meaning that on average 80 rooms were occupied each night. Calculate the Occupancy Percentage: Occupancy Percentage = Number of occupied rooms / Total number of rooms x 100 Calculate the Room Revenue: Room Revenue = ADR x Number of Rooms Occupied Calculate the Average Daily Rate (ADR): ADR = Total Room Revenue / Number of Rooms Occupied Calculate the Revenue per available room (Rev PAR): RevPAR = ADR x Occupancy Percentage


Answer:

Using the given information:

Number of occupied rooms = 80

Total number of rooms = 100

ADR = Php6,500

To calculate the Occupancy Percentage:

Occupancy Percentage = Number of occupied rooms / Total number of rooms x 100

Occupancy Percentage = 80 / 100 x 100

Occupancy Percentage = 80%

To calculate the Room Revenue:

Room Revenue = ADR x Number of Rooms Occupied

Room Revenue = Php6,500 x 80

Room Revenue = Php520,000

To calculate the Average Daily Rate (ADR):

ADR = Total Room Revenue / Number of Rooms Occupied

ADR = Php520,000 / 80

ADR = Php6,500

To calculate the Revenue per available room (RevPAR):

RevPAR = ADR x Occupancy Percentage

RevPAR = Php6,500 x 80%

RevPAR = Php5,200
