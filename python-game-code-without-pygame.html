7. pls help me find the code.Create a Python program that prompts the user to enter a number and prints out the multiplication table for that number from 1 to 10. ​


Answer:

Here's a Python program that prompts the user to enter a number and prints out the multiplication table for that number from 1 to 10:

```python

number = int(input("Enter a number: "))

print("Multiplication Table for", number)

print("----------------------------")

for i in range(1, 11):

product = number * i

print(number, "x", i, "=", product)

```

In this program, we use the `input()` function to prompt the user to enter a number. The entered value is converted to an integer using `int()` and stored in the `number` variable.

We then use a `for` loop to iterate from 1 to 10. In each iteration, we calculate the product of the `number` and the current loop variable (`i`), and store it in the `product` variable. Finally, we print out the multiplication table by displaying the equation `number x i = product` for each iteration.

You can run this program, enter a number when prompted, and it will display the multiplication table for that number from 1 to 10.

13. 2. A python curls up to touch the tip of its own tail with its nose, forming the shape of a circle. The python is 2.6π meters long. What is the radius of the circle that the python forms?


Answer:

1.3 meters

Step-by-step explanation:

The said length of the python (2.6π meters) is also the circumference.

We will just derive to get the radius.

Circumference = 2π(radius) or Circumference = πd

2.6π = 2πr

2.6π/2π = 2πr/2π

1.3 = r

radius is 1.3 meters
