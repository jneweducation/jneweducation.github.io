To determine what type are sentences above, let’s discuss more about conditional sentence.

What is a Conditional Sentence?

Conditional sentences are statements that are used to state hypothetical situations and their consequences.

What Are the Types of Conditional Sentences?

There are four types of conditional sentences: Type-0, type-1, type-2, and type-3

1. Type-0 (Zero conditional sentences)

This type expresses general truth or situations.

Structure: If + Simple present, Simple present

Example: If you eat too much candy, you get cavities

2. Type-1 (First conditional sentences)

This type expresses something that would like to happen (but not guaranteed)

Structure: If + simple present, Subject + will + V1

Example: If you go to school later, your father will drive you

3. Type-2 (Second conditional sentences)

This type expresses something unrealistic or will not likely happen in the future.

Structure: If + simple past, Subject + would + V1

Example: If I were you, I would punch him in the face

4. Type-3 (Third conditional sentences)

This type expresses present circumstances would be different if something different had happened in the past.

Structure: If + Past Perfect tense, Subject + would+have+V3

Example: If I had proposed to you, you would not have married him.

180 minutes × 3 pages / 20 minutes = 27 pages/minute
