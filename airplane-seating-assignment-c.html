1. There are 400 seats in the airplane . 80% of the seats are economy class . How many seats are there in the economy class?


Answer:

368 seats

Step-by-step explanation:

400× 0.08= 32

400-32= 368

Answer:

Step-by-step explanation:

Given:

Total number of seats = 400

Number of economy seats = 80%

Find the number of economy seats:

[tex]\text{Number of economy seats = } 80\% \text { of } 400[/tex]

[tex]\text{Number of economy seats = } \dfrac{80}{100} \times 400[/tex]

[tex]\text{Number of economy seats = } 80 \times 4[/tex]

[tex]\text{Number of economy seats = }320[/tex]

Answer: There are 320 economy seats.
