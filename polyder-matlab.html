1. What is the meaning of MATLAB?​


Answer:

Matrix Laboratory

Explanation:

The name MATLAB stands for matrix laboratory. MATLAB was originally written to provide easy access to matrix software developed by the LINPACK and EISPACK projects, which together represent the state-of-the-art in software for matrix computation. MATLAB has evolved over a period of years with input from many users.

4. Create a MATLAB code that solves roots of non-linear equations using MOSS Method. The code should have the following features: 1. The nonlinear function, manipulated to the form ₁+1=8(x) can be declared directly on the MLX file. 2. Accepts an initial estimate and the error tolerance. 3. Checks if the initial estimate is a root of the inputted function. 4. Displays the iteration table. (For more information, watch the Week 3 Lecture Video: ALTERNATIVE CODING FOR BISECTION METHOD). 5. Gives the following output values: approximate root, percent approximate relative error, number of iterations required to meet the condition (percent approximate relative error


Here is a sample MATLAB code that solves for the roots of nonlinear equations using the MOSS method:

% Define the function

f = @(x) 8*x - 1;

% Define the initial estimate and error tolerance

x0 = 1;

tol = 1e-6;

% Check if the initial estimate is a root of the function

if abs(f(x0)) < tol

fprintf('Initial estimate is already a root of the function\n');

else

% Initialize variables for keeping track of the iterations

x1 = x0;

err = 1;

i = 1;

% Iterate until the error tolerance is met

while err > tol

% Calculate the next estimate using the MOSS method

x2 = (f(x1) * x0 - f(x0) * x1) / (f(x1) - f(x0));

% Calculate the relative error

err = abs((x2 - x1) / x2);

% Update the variables for the next iteration

x0 = x1;

x1 = x2;

i = i + 1;

end

% Display the iteration table

fprintf('Iteration\tx0\tx1\tError\n');

for j = 1:i-1

fprintf('%d\t\t%.6f\t%.6f\t%.6f\n', j, x0, x1, err);

end

% Print the output values

fprintf('\nApproximate root: %.6f\n', x2);

fprintf('Percent approximate relative error: %.6f%%\n', err*100);

fprintf('Number of iterations required: %d\n', i-1);

end

This code defines the function as a function handle, accepts an initial estimate and error tolerance as input, and then iteratively applies the MOSS method until the error tolerance is met. It also displays the iteration table and prints the approximate root, percent approximate relative error, and number of iterations required as output.
