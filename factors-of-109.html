Answer:

D. 109

Step-by-step solving: Evaluating Discriminant

3x² + 5x - 7 = 0

a = 3     b = 5     c = -7

D = b² - 4ac

D = 5² - 4(3)(-7)

D = 25 + 84

D = 109

2a + 7 + 109 = 180

First, we simplify the expression inside the parentheses:

2(32) + 7 + 109 = 180

This simplifies to:

64 + 7 + 109 = 180

Next, we perform addition:

64 + 7 + 109 = 180

This simplifies to:

180 = 180
