9. Formula for permulation p(n,r)​


Answer:

Formula and Calculation of Permutation

The formula for a permutation is: P(n,r) = n! / (n-r)! The generalized expression of the formula is, "How many ways can you arrange 'r' from a set of 'n' if the order matters?" A permutation can be calculated by hand as well, where all the possible permutations are written out.

18. How many permulations are there in the letters of the word LOVER?​


Answer:

24

Step-by-step explanation:

1st position: you have 4 choices { L, O, V, E }, pick one letter { E }

2nd position: you have 3 remaining choices { L, O, V }, pick one { L }

3rd position: 2 remaining choices {O, V}, pick one { O }

4th position: 1 remaining choice { V }

Continuing the same pattern, you will arrive at 4 x 3 x 2 x 1 arrangements.

4P4 = 4! = 4 x 3 x 2 x 1 = 24
