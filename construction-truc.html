15. 4. Which statement is NOT truc?A. Every counting number is a whole number.B. Every integer is a rational number.C. Every decimal is an irrational number.D. Every natural number is a real number.C. Every demical is an irrational number.(´∩｡• ᵕ •｡∩`) Sana makatulong ෆ╹ .̮ ╹ෆ​


Answer:

Every decimal is an irrational number. (C)

Step-by-step explanation:

Observe that:

Option A is always true because counting numbers are 1, 2, 3, and so on which are whole numbers.

Option B is also always true because every integer can be expressed as a fraction which means it is a rational number.

Option C is false because not all decimals are irrational, terminating and repeating decimals can be expressed as a fraction which means they are rational. Meanwhile, non-terminating decimals are irrational numbers.

Option D is also always true because natural numbers are part of real numbers which includes both rational and irrational numbers.

Thus, the correct answer is option C.
