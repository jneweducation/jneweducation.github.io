3. The average of 3 numbers is 33. The first number is twice the second number and the third number is one-third of the first number​


Answer:

The numbers are 54, 27 and 18

Step-by-step explanation:

Let x = the second number

Let 2x = the first number

Let 2x/3 = the third number

(2x + x + 2x/3) / 3 = 33 is used to represent the average

Divide both sides by 3 to get rid of the fraction

(3)(2x + x + 2x/3) / 3 = 33(3)
(6x + 3x +2x) / 3 = 99

Simplify the left side by adding 6x + 3x + 2x

11x / 3 = 99

Multiply both sides by 3/11

x = 27

Since x is used to represent the second number, the second number is 27.

Substitute the value of x to the other expressions to get the other numbers:

first number = 2x

2(27) = 54

second number = 2x/3

2(27) = 54

54/3 = 18

The three numbers are 54, 27 and 18.

29. The first numbers is more than the third by one half of the second.the sum of the second and third numbers is one more than the first. If the second is subtracted from the sum of the first and third numbers the result is 10. What are the numbers?


a = first number

b = second number

c = third number


[tex]a=c+\frac{1}{2} (b)[/tex]   (eq.1)

[tex]b+c=a+1[/tex]       (eq.2)

[tex](a+c)-b=10[/tex]     ( eq.3)

from eq.1

              b = 2a - 2c

              b = 2(a - c)

         a - c = b/2  ( eq.4)

from eq.2:

           a - c = b - 1 (eq.5)

equate eq.4 and eq5:

                                 a - c = a - c

                                   b/2 = b - 1

                                       b = 2b - 2

                                       b = 2

substitute value of b to eq.1:

                                           a = c + 1/2(2)

                                           a = c + 1

then substitute this value of a to eq.3:

                                           (a + c) - b = 10

                                      (c + 1 + c) - 2 = 10

                                                       2c = 11

                                                         c = 11/2

substitute the values of b and c to eq.1 to solve for a:

                  a = 11/2 + 1/2(2)

                  a = 13/2
