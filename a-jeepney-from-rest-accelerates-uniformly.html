1. a jeepney from the rest accelerates uniformly over a time of 638 seconds.covers a distace of 18.25m determine the acceleration of the jeepney?


ACCELERATION is the rate of change of velocity over a period of time. Thus, if we are to put it on a mathematical equation, then it would be

a = (change in velocity)/time

a = (final velocity - initial velocity)/ time

a = (Vf - Vi)/ t

If we are to inspect the given problem, it states that the jeepney has started moving from rest. Thus, its initial velocity is zero (0). Why?

Because an object which is not moving or is at rest has a velocity of zero (0).

Recall that the FORMULA IN FINDING FOR VELOCITY is

V = distance / time

V = d/t

Thus, if the object in not moving (no distance traveled or covered), then velocity is zero.

Note that we are also given a time by which the jeepney has moved to find for its acceleration, and a distance it traveled during that time.

So we make use of the KINEMATICS FORMULAS.

d = (Vo)(t) + (1/2)(a)(t^2);

(Vf)^2 = (Vo)^2 + at; and

Vf = Vo + 2ad

Note that if we use the first equation, then we can directly solve for the acceleration.

d = (Vo)(t) + (1/2)(a)(t^2)

18.25m = (0)(638s) + (1/2)(a)(638s)^2

18.25 = 0 + 203,522 a

a = 8.9671 x 10^-5

a = 0.000080671 m/sec^2

Therefore, the acceleration is 0.000080671 m/sec^2

3. A jeepney from rest accelerates uniformly over a time of 8.4 seconds and covers a distance of 21 meters.Determine the acceleration of the jeepney


The general formula for finding the distance given the velocity, acceleration and time is:
[tex]x = v_{o}t \: + \frac{1}{2} {at}^{2} [/tex]
The initial velocity can be assumed to be zero since it is stated in the problem that the jeepney accelrates from rest.

Since we already have the value for distance, initial velocity and time, thus the only missing value is acceleration. We then have to transform the above equation into our working equation, that is,
[tex]a = \frac{2x}{{t}^{2} } - \frac{2v_{o}}{t} [/tex]
Substituting the given values into the equation,
[tex]a = \frac{2(21m)}{{8.4s}^{2} } - \frac{2(0)}{8.4s} [/tex]
[tex]a = 0.5952 \frac{m}{{s}^{2} } [/tex]
[tex]a = 0.60 \frac{m}{{s}^{2} } [/tex]
The acceleration of the jeepney then is 0.60 meters per second squared.

12. A jeepney from rest accelerates uniformly over a time 3.25 seconds and covers 9 distance of 15m. Determine the accelerations of the jeepneyGIVEN:ASK:FORMULA:​


Answer:

Given:

Initial velocity, v_i = 0 m/s (since the jeepney starts from rest)

Time taken, t = 3.25 s

Distance covered, d = 15 m

Ask:

Acceleration of the jeepney

Formula:

We can use the following kinematic equation to calculate the acceleration of the jeepney:

d = v_i*t + (1/2)at^2

where d is the distance covered, v_i is the initial velocity, t is the time taken, and a is the acceleration.

We can rearrange the equation to solve for a:

a = 2*(d - v_i*t) / t^2

Substituting the given values, we get:

a = 2*(15 - 0*3.25) / (3.25)^2

Simplifying the expression, we get:

a = 1.41 m/s^2

Therefore, the acceleration of the jeepney is 1.41 m/s^2.

22. A jeepney from rest accelerates uniformly over a time of 3.25 seconds and covers a distance of 15m. Determine the acceleration of the jeepney?


Solution:

The second kinematic equation of horizontal rectilinear motion is described by the formula

d = (vi)(t) + (1/2)(at²)

where:

d = distance

vi = initial velocity

t = time

a = acceleration

Listing the given values

t = 3.25 s

d = 15 m

vi = 0 (from rest)

Solving for a

d = (vi)(t) + (1/2)(at²)

15 m = (0)(3.25 s) + (1/2)(a)(3.25 s)²

15 m = 0 + (5.28125 s²)(a)

15 m = (5.28125 s²)(a)

a = 15 m / 5.28125 s²

a = 2.84 m/s²

Therefore, the acceleration of the jeepney is 2.84 m/s².
