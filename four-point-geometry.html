1. Give four descriptions of a point. ( geometry )​


Hi! Before you copy paste, If you find my response helpful and informative, I would greatly appreciate it if you could select it as the "Brainliest" answer.



Answer:
A point in geometry can be described in different ways. Here are four possible descriptions:

1. A point is a location in space that has no size, shape or dimension. It is represented by a dot or a cross on a piece of paper.

2. A point is defined by its coordinates in a coordinate system. In a two-dimensional plane, a point is represented by an ordered pair of numbers (x,y), where x is the horizontal coordinate and y is the vertical coordinate.

3. A point is a basic element of geometry that is used to define other geometric objects such as lines, planes, and angles. A line is defined as a set of points that extend infinitely in both directions, while a plane is defined as a flat surface that extends infinitely in all directions.

4. A point can be located relative to other points or objects in space. For example, a point can be inside, outside, or on the boundary of a shape. It can also be used to describe the position of an object in space, such as the location of a star in the sky.

8. can 2 points create a plane in geometry


Answer:

If two lines intersect, then their intersection is exactly one point. Through any three non-collinear points, there exists exactly one plane. A plane contains at least three non-collinear points. If two points lie in a plane, then the line containing them lies in the plane.

In modern mathematics, a point refers more generally to an element of some set called a space. Being a primitive notion means that a point cannot be defined in terms of previously defined objects.
