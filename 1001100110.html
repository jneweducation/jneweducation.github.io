11. equivalent of 10110110


Answer:

(10110110)2 = (182)10

Explanation:

Step 1: Write down the binary number:

10110110

Step 2: Multiply each digit of the binary number by the corresponding power of two:

1x27 + 0x26 + 1x25 + 1x24 + 0x23 + 1x22 + 1x21 + 0x20

Step 3: Solve the powers:

1x128 + 0x64 + 1x32 + 1x16 + 0x8 + 1x4 + 1x2 + 0x1 = 128 + 0 + 32 + 16 + 0 + 4 + 2 + 0

Step 4: Add up the numbers written above:

128 + 0 + 32 + 16 + 0 + 4 + 2 + 0 = 182.

So, 182 is the decimal equivalent of the binary number 10110110.
