18. A craft fair needs booths for 50 vendors. Each row can have up to 6 booths. The fair will put as many booths in each row as possible. How many rows of booths will there be? please help! answers 6 rows 7 rows 8 rows 9 rows


The number of booths needed can be found by dividing the total number of vendors by the number of booths in each row:

50 vendors / 6 booths per row = 8.33 rows

Since we cannot have a fraction of a row, we need to round up to the nearest whole number of rows. Therefore, we will need 9 rows of booths to accommodate all 50 vendors.

So the answer is 9 rows.
