10. sino si juan luna at antonio luna


Answer:

Si Juan Luna y Novicio ang nagpinta ng pamosong larawan na “Spoliarium”. Siya ay nakilala sa buong mundo sa pamamagitan ng kanyang pinsel gaya ng pagkakilala sa kanyang mga kaibigan sa pamamagitan ng pluma at espada.

Si Antonio Luna (29 Oktubre 1866 - 7 Hunyo 1899) ay isang Pilipinong parmasiyotiko at isang heneral na lumaban sa Digmaang Pilipino-Amerikano. Siya rin ang nagtatag ng kauna-unahang akademyang militar sa Pilipinas, na naitatag noong Unang Republika ng Pilipinas.

Algorithm:

BubbleSort (Arr, N) // Arr is an array of size N.

{

For ( I:= 1 to (N-1) ) // N elements => (N-1) pass

{

// Swap adjacent elements of Arr[1:(N-I)]such that

// largest among { Arr[1], Arr[2], ..., Arr[N-I] } reaches to Arr[N-I]

For ( J:= 1 to (N-I) ) // Execute the pass

{

If ( Arr [J] > Arr[J+1] )

Swap( Arr[j], Arr[J+1] );

}

}

}

18. What is sort ascending and sort descending


Ascending denotes smallest to largest, 0 to 9, and/or A to Z, whereas Descending denotes largest to smallest, 9 to 0, and/or Z to A. The smallest, first, or earliest item in the order will appear at the top of the list in ascending order: for numbers or amounts, the sort is smallest to largest.
