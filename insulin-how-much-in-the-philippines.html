if an insulin injection contains 100 units of insulin injections in each millilitres, how many millilitre should be injected to receive 40 units of insulin?

To find the number of millilitres of insulin that should be injected to receive 40 units of insulin, we need to divide the number of units of insulin we want to inject by the number of units per millilitre. We can use the formula:

number of millilitres = number of units of insulin / units of insulin per millilitre

In this case, we can plug in the values to get:

number of millilitres = 40 units of insulin / 100 units of insulin per millilitre

Dividing, we get:

number of millilitres = 0.4 millilitres

Therefore, to receive 40 units of insulin, the person should inject 0.4 millilitres of insulin.
