19. The length of a wall is seven meters longer than its width, and its area is 30m^2. What is the equation represent the given situation?


If you're asking about the equation:

w (w + 7) = 30

If you're asking about its dimensions:

w (w + 7) = 30

w² + 7w = 30

w² + 7w - 30 = 0

w² + 10w - 3w - 30 = 0

w (w + 10) - 3 (w + 10) = 0

(w + 10)(w - 3) = 0

w + 10 = 0, w - 3 = 0

w₁ = -10, w₂ = 3

Length = w₂ + 7 , Width = w₂

Length = 3 + 7 , Width = 3

Length = 10 , Width = 3

Therefore, its length is 10m while its width is 3m.
