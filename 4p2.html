16. If (2p – 5)(a) = 4p2 – 25, then which of the following is the value of a?


Answer: The value of a = 2p+5

Step-by-step explanation:

(2p – 5)(a) = 4p2 – 25

First step: Find the factor of 4p² – 25

Write 4p² as (2p)² → (2p)² – 25

Rewrite 25 as 5² → (2p)² – 52

Since both terms are perfect squares, factor them using the difference formula of twos, a²– b² = ( a + b ) ( a – b ) where a = 2p and b = 5 → ( 2p + 5 ) ( 2p – 5 )

Thus, 4p² – 25 =( 2p + 5 ) ( 2p − 5 )

Second step:

(2p – 5) (a) = 4p² – 25

Then, plug in the factors from 4p² – 25 into the equation → (2p – 5) (a) = (2p-5) (2p+5)

Move (2p - 5) that is on the left side to the right side and place it under (2p-5) (2p+5) so that it forms a division → a = (2p-5)(2p+5) / (2p – 5)

Because the right-hand side has the same factors so that these factors can be crossed out or omitted → a = 2p + 5

Thus, the equation (2p – 5) (a) = 4p² – 25 has a value of a = 2p + 5.

[tex]\large\underline{\mathbb{SOLUTION}:}[/tex]

A quadratic equation is written in the form ax²+bx+c = 0. Use the quadratic formula to solve for the given:

[tex]\boxed{\sf{x = \dfrac{\text{-}\red{b} \pm \sqrt{\red{b}^{2} - 4\orange{a}\purple{c}} }{2 \orange{a}} \rightarrow (where \ \orange{a} \neq 0)}}[/tex]

[tex]\sf{p = \dfrac{\text{-}\red{4} \pm \sqrt{\red{4}^{2} - 4(\orange{4})(\purple{1})} }{2 (\orange{4})} }[/tex]

[tex]\sf{p = \dfrac{\text{-}\red{4} \pm \sqrt{\red{4}^{2} - 16(\purple{1})} }{2 (\orange{4})} }[/tex]

[tex]\sf{p = \dfrac{\text{-}\red{4} \pm \sqrt{16- 16} }{8} }[/tex]

[tex]\sf{p = \dfrac{\text{-}\red{4} \pm \sqrt{0} }{8} }[/tex]

[tex]\sf{p = \dfrac{\text{-}\red{4} + \sqrt{0} }{8} } \: \: \: \: \: \: \: \: \: \: \: \: or \: \: \: \: \: \: \: \: \: \: \: \: \sf{p = \dfrac{\text{-}\red{4} - \sqrt{0} }{8} }[/tex]

Since the two solutions are just the same, they should be written as a single solution. Hence,

[tex]\underline{\boxed{\sf{p = \dfrac{\text{-}1}{2} } }}[/tex]
