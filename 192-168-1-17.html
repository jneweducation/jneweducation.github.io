4. Find the missing number in the following series: 4 ? 17 51 56 168


Answer:

The missing number is 12.

4, 12, 17, 51, 56, 168

Step-by-step explanation:

Number Pattern

Number pattern is a series of numbers that follow a certain sequence or rule. There is no specific formula that we can use to complete a pattern. It is more on analyzing the given numbers.

To find the missing number in the pattern above, we need to find what is common with the numbers and come up with a specific rule.

We can easily determine the rule by finding the relationship between two numbers that are closed to each other.

4, ?, 17, 51, 56, 168

Since the second number is missing, let us start by finding the relationship between numbers from the third number.

17, 51

(17 × 3 = 51)

For these numbers, we can say that we need to multiply the number by three to get the next number.

51, 56

(51 + 5 = 56)

We need to add five to get the next number.

56, 168

(56 × 3 = 168)

Multiply the number by three again.

Therefore, the pattern rule is × 3, + 5.

Apply the rule.

4 (× 3), 12 (+ 5), 17 (× 3), 51 (+ 5), 56 (× 3), 168

= 4, 12, 17, 51, 56, 168

5. The product of two positive consecutive even integer is 168. Assuming the smaller integer to be x, frame an equation for the statement. A. x (x + 2) = 168 B. x2 = 168 C. 2x = 168 D. x (2x) = 168​


Answer:

A. x(x+2) = 168

Step-by-step explanation:

The product of two positive consecutive even integer is 168.

Smaller even integer: xSecond even integer, add 2: (x + 2)Product: 168Operation: Multiplication

Equation:

(x)(x+2) = 168

To solve for x, the first even integer:

x² + 2x - 168 = 0

Factor:

(x + 14)(x - 12) = 168

Equate each factor to 0:

x + 14 = 0

x = -14

x - 12 = 0

x = 12

Smaller even integer x: 12

Second even integer: 12 + 2 = 14

The consecutive even  integers are 12 and 14.

Check:

(12)(14) = 168

168 = 168  (True)
