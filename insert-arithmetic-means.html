6. Find the indicated means of the following arithmetic sequence. 1. Insert 3 arithmetic means between 3 and 39.2. Insert 2 Arithmetic means between 69 and - 12.3. Insert 3 arithmetic means between -65 and 35​


✏️ARITHMETIC MEAN===============================

- Find the indicated means of the following arithmetic sequence.

1. Insert 3 arithmetic means between 3 and 39.

- Let 3 be the first term. The 3 arithmetic means would be 2nd, 3rd, and 4th term, and the 5th term would be 39. Identify the common difference by using the nth term of the Arithmetic Sequence Formula.

[tex]\begin{gathered} \begin{aligned}& \bold{ \color{lightblue}Formula:} \\ & \boxed{a_n= a_1 + (n - 1)d } \end{aligned} \end{gathered}[/tex]

[tex]39 = 3 + (5 - 1)d[/tex]

[tex]39 - 3 = 5d - d[/tex]

[tex]36 = 4d[/tex]

[tex] \begin{gathered} \frac{36}{4} = \frac{ \cancel4d}{ \cancel4} \end{gathered}[/tex]

[tex]d = 9[/tex]

- Find the 3 arithmetic means with a common difference of 9.

[tex]a_2 = a_1 + d = 9 + 3 = 12[/tex]

[tex]a_3 = a_2 + d = 12 + 9= 21[/tex]

[tex]a_4 = a_3 + d = 21 + 9= 30[/tex]

- Therefore, the 3 arithmetic means are:

[tex] \large \boxed{ \sf \green{ \: 12, 21, and \: 30 \: }}[/tex]===============================

2. Insert 2 Arithmetic means between 69 and - 12

- Let 69 be the first term. The 2 arithmetic means would be 2nd, and 3rd term, and the 4th term would be -12. Identify the common difference by using the nth term of the Arithmetic Sequence Formula.

[tex]\begin{gathered} \begin{aligned}& \bold{ \color{lightblue}Formula:} \\ & \boxed{a_n= a_1 + (n - 1)d } \end{aligned} \end{gathered}[/tex]

[tex] \text - 12 = 69 + (4 - 1)d[/tex]

[tex] \text - 12 - 69 + 4d - d[/tex]

[tex] \text -81 = 3d[/tex]

[tex]\begin{gathered} \frac{ \text - 81}{3} = \frac{ \cancel3d}{ \cancel3} \end{gathered}[/tex]

[tex]d = \text - 27[/tex]

- Find the 2 arithmetic means with a common difference of -27.

[tex]a_2 = a_1 + d = 69 + (\text -27) = 42[/tex]

[tex]a_3 = a_2 + d = 42 + (\text -27) = 15[/tex]

- Therefore, the 2 arithmetic means are:

[tex]\large \boxed{ \sf \green{ \: 42 \: and \: 15\:}}[/tex]===============================

3. Insert 3 arithmetic means between -65 and 35

- Let -65 be the first term. The 3 arithmetic means would be 2nd, 3rd, and 4th term, and the 5th term would be 35. Identify the common difference by using the nth term of the Arithmetic Sequence Formula.

[tex]\begin{gathered} \begin{aligned}& \bold{ \color{lightblue}Formula:} \\ & \boxed{a_n= a_1 + (n - 1)d } \end{aligned} \end{gathered}[/tex]

[tex]35 = \text - 65 + (5 - 1)d[/tex]

[tex]35 + 65= + 5d - d[/tex]

[tex]100= 4d[/tex]

[tex]\begin{gathered} \frac{100}{4} = \frac{ \cancel4d}{ \cancel4} \end{gathered}[/tex]

[tex]d = 25[/tex]

- Find the 3 arithmetic means with a common difference of 25.

[tex]a_2 = a_1 + d = \text -65 + 25 = \text - 40[/tex]

[tex]a_2 = a_1 + d = \text -40 + 25 = \text - 15[/tex]

[tex]a_2 = a_1 + d = \text -15 + 25 =10[/tex]
